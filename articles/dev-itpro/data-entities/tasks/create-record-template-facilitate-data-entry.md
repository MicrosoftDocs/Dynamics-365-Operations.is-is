<span data-ttu-id="0802d-104">Þetta ferli sýnir hvernig stofna færslusniðmát til að gildi í svæði sem oft notað ekki þurfa að slá inn sérstaklega fyrir hverja ný færsla.</span><span class="sxs-lookup"><span data-stu-id="0802d-104">This procedure demonstrates how to create a record template so that field values that are used often do not have to be entered explicitly for each new record.</span></span> <span data-ttu-id="0802d-105">Í þessu ferli er stofnað ný færsla fyrir nýjar fartölvur sem ætti að bæta við þínar eignir.</span><span class="sxs-lookup"><span data-stu-id="0802d-105">In this procedure, you’ll create a new record for new laptops that should be added to your fixed assets.</span></span> <span data-ttu-id="0802d-106">Þetta ferli notar USMF sýnifyrirtæki.</span><span class="sxs-lookup"><span data-stu-id="0802d-106">This procedure uses the USMF sample company.</span></span>

4. <span data-ttu-id="0802d-110">Í reitinn Heiti skal slá inn gildi.</span><span class="sxs-lookup"><span data-stu-id="0802d-110">In the Name field, type a value.</span></span>

5. <span data-ttu-id="0802d-112">Í svæði Leita að nafni skal slá inn gildi.</span><span class="sxs-lookup"><span data-stu-id="0802d-112">In the Search name field, type a value.</span></span>
    * <span data-ttu-id="0802d-113">Til dæmis slá inn „fartölva“.</span><span class="sxs-lookup"><span data-stu-id="0802d-113">For example, enter 'laptop.'</span></span>
