# <a name="set-up-withholding-tax-component-groups-for-the-tds-tax-type"></a><span data-ttu-id="c13e8-103">Setja upp íhlutaflokka staðgreiðsluskatts fyrir TDS-skattgerðir</span><span class="sxs-lookup"><span data-stu-id="c13e8-103">Set up withholding tax component groups for the TDS tax type</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="c13e8-104">Í þessu efnisatriði er útskýrt hvernig á að setja upp hlutahópa staðgreiðsluskatts, svo sem **Leiga** og **Verktaki** fyrir skatt sem er dreginn frá upprunalegri greiðslu (TDS).</span><span class="sxs-lookup"><span data-stu-id="c13e8-104">This topic explains how to set up withholding tax component groups, such as **Rent** and **Contractor**, for the Tax Deducted at Source (TDS) tax type.</span></span>

1. <span data-ttu-id="c13e8-105">Farið í **Skattur \> Uppsetning \> Staðgreiðsluskattur \> Íhlutaflokkar staðgreiðsluskatts**.</span><span class="sxs-lookup"><span data-stu-id="c13e8-105">Go to **Tax \> Setup \> Withholding tax \> Withholding tax component groups**.</span></span>

    <span data-ttu-id="c13e8-106">[![Síða staðgreiðsluskattshlutaflokka](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)</span><span class="sxs-lookup"><span data-stu-id="c13e8-106">[![Withholding tax component groups page](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)</span></span>

2. <span data-ttu-id="c13e8-107">Í reitnum **Skattgerð** skal velja **TDS** til að setja upp íhlutaflokka staðgreiðsluskatts fyrir skattgerð TDS.</span><span class="sxs-lookup"><span data-stu-id="c13e8-107">In the **Tax type** field, select **TDS** to set up withholding tax component groups for the TDS tax type.</span></span>
3. <span data-ttu-id="c13e8-108">Veldu **Nýtt** á aðgerðasvæðinu til að búa til línu.</span><span class="sxs-lookup"><span data-stu-id="c13e8-108">On the Action Pane, select **New** to create a line.</span></span>
4. <span data-ttu-id="c13e8-109">Í reitnum **Hópur staðgreiðsluskattshluti** skal færa inn heiti fyrir hóp TDS-hluta.</span><span class="sxs-lookup"><span data-stu-id="c13e8-109">In the **Withholding tax component group** field, enter a name for the TDS component group.</span></span>
5. <span data-ttu-id="c13e8-110">Í reitnum **Lýsing** skal færa inn lýsingu.</span><span class="sxs-lookup"><span data-stu-id="c13e8-110">In the **Description** field, enter a description.</span></span>
6. <span data-ttu-id="c13e8-111">Á flýtiflipanum **Almennt** í reitnum **Staða** skal velja búsetustöðu TDS íhlutahópsins.</span><span class="sxs-lookup"><span data-stu-id="c13e8-111">On the **General** FastTab, in the **Status** field, select the residential status of the TDS component group.</span></span> <span data-ttu-id="c13e8-112">Valkostirnir eru **Ríkisborgari** og **Ekki ríkisborgari**.</span><span class="sxs-lookup"><span data-stu-id="c13e8-112">The options are **Resident** and **Non-resident**.</span></span>
7. <span data-ttu-id="c13e8-113">Í reitinn **Kaflakóði** er færður inn kaflakóðinn sem er lagður á TDS-hlutaflokkinn.</span><span class="sxs-lookup"><span data-stu-id="c13e8-113">In the **Section code** field, enter the section code that is applied to the TDS component group.</span></span>
8. <span data-ttu-id="c13e8-114">Lokið síðunni.</span><span class="sxs-lookup"><span data-stu-id="c13e8-114">Close the page.</span></span>
