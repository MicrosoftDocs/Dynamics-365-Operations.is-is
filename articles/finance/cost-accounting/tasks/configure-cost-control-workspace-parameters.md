# <a name="configure-cost-control-workspace-parameters"></a><span data-ttu-id="5ce54-103">Skilgreina færibreytur vinnusvæðis kostnaðarstýringar</span><span class="sxs-lookup"><span data-stu-id="5ce54-103">Configure cost control workspace parameters</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="5ce54-104">Nota þetta ferli til að grunnstilla Vinnusvæði kostnaðarstýringar svo stjórnendur á ýmsum stigum í fyrirtæki geti fengið innsýn inn í kostnaðarhluta þess, eins og og kostnaðarstaði og vöruflokka.</span><span class="sxs-lookup"><span data-stu-id="5ce54-104">Use this procedure to configure the Cost control workspace so that managers at various levels in an organization can gain insight into their cost objects, such as cost centers and product groups.</span></span> <span data-ttu-id="5ce54-105">USP2 sýnifyrirtækið var notað til að stofna þessa skráningu.</span><span class="sxs-lookup"><span data-stu-id="5ce54-105">The USP2 demo company was used to create this recording.</span></span>

1. <span data-ttu-id="5ce54-106">Fara í kostnaðarbókhald > Uppsetning > kostnaðarstýring vinnusvæði skilgreining.</span><span class="sxs-lookup"><span data-stu-id="5ce54-106">Go to Cost accounting > Setup > Cost control workspace configuration.</span></span>
2. <span data-ttu-id="5ce54-107">Smellið á „Nýtt“.</span><span class="sxs-lookup"><span data-stu-id="5ce54-107">Click New.</span></span>
3. <span data-ttu-id="5ce54-108">Í reitinn Heiti skal slá inn gildi.</span><span class="sxs-lookup"><span data-stu-id="5ce54-108">In the Name field, type a value.</span></span>
4. <span data-ttu-id="5ce54-109">Sláið inn gildi í reitnum „Lýsing“.</span><span class="sxs-lookup"><span data-stu-id="5ce54-109">In the Description field, type a value.</span></span>
5. <span data-ttu-id="5ce54-110">Veljið Já í svæðinu Útgefið.</span><span class="sxs-lookup"><span data-stu-id="5ce54-110">Select Yes in the Published field.</span></span>
    * <span data-ttu-id="5ce54-111">Ef þú velur Já, geta notendur sem er úthlutað einu af eftirfarandi hlutverkum séð skýrsluna í Vinnusvæði kostnaðarstýringar: Stjórnandi kostnaðarbókhalds, endurskoðandi kostnaðar, afgreiðslustarfsmaður endurskoðanda kostnaðar, og eftirlitsmaður kostnaðarhluta.</span><span class="sxs-lookup"><span data-stu-id="5ce54-111">If you set this option to Yes, users who are assigned one of these roles can see the report in the Cost control workspace: cost accounting manager, cost accountant, cost accountant clerk, or cost object controller.</span></span> <span data-ttu-id="5ce54-112">Ef þú velur Nei, geta aðeins notendur sem er úthlutað einu af eftirfarandi hlutverkum séð skýrsluna í Vinnusvæði kostnaðarstýringar: Stjórnandi kostnaðarbókhalds, endurskoðandi kostnaðar, eða afgreiðslustarfsmaður endurskoðanda kostnaðar.</span><span class="sxs-lookup"><span data-stu-id="5ce54-112">If you set this option to No, only users who are assigned one of these roles can see the report in the Cost control workspace: cost accounting manager, cost accountant, or cost accountant clerk.</span></span>  
6. <span data-ttu-id="5ce54-113">Útvíkka Afmörkun gagna svæðið.</span><span class="sxs-lookup"><span data-stu-id="5ce54-113">Expand the Data filtering section.</span></span>
7. <span data-ttu-id="5ce54-114">Í reitinn Kostnaðarstýringareining skal slá inn eða velja gildi.</span><span class="sxs-lookup"><span data-stu-id="5ce54-114">In the Cost control unit field, enter or select a value.</span></span>
8. <span data-ttu-id="5ce54-115">Í svæði Upphafleg útgáfa kostnaðaráætlunar slá inn eða velja gildi.</span><span class="sxs-lookup"><span data-stu-id="5ce54-115">In the Budget original version field, enter or select a value.</span></span>
9. <span data-ttu-id="5ce54-116">Sláið inn eða veljið gildi í reitnum Stigveldi víddar kostnaðareiningar.</span><span class="sxs-lookup"><span data-stu-id="5ce54-116">In the Cost element dimension hierarchy field, enter or select a value.</span></span>
10. <span data-ttu-id="5ce54-117">Sláið inn eða veljið gildi í reitnum Stigveldi víddar kostnaðarhlutar.</span><span class="sxs-lookup"><span data-stu-id="5ce54-117">In the Cost object dimension hierarchy field, enter or select a value.</span></span>
11. <span data-ttu-id="5ce54-118">Útvíkka hlutann Úthluta útreikningsfærslum.</span><span class="sxs-lookup"><span data-stu-id="5ce54-118">Expand the Assign calculation records section.</span></span>
12. <span data-ttu-id="5ce54-119">Smellið á Nýtt.</span><span class="sxs-lookup"><span data-stu-id="5ce54-119">Click New.</span></span>
13. <span data-ttu-id="5ce54-120">Í listanum skal merkja valda línu.</span><span class="sxs-lookup"><span data-stu-id="5ce54-120">In the list, mark the selected row.</span></span>
14. <span data-ttu-id="5ce54-121">Sláið inn eða veldu gildi í reitnum Fjárhagsdagatal tímabil.</span><span class="sxs-lookup"><span data-stu-id="5ce54-121">In the Fiscal calendar period field, enter or select a value.</span></span>
15. <span data-ttu-id="5ce54-122">Í svæði Raunveruleg útgáfa, slá inn eða velja gildi.</span><span class="sxs-lookup"><span data-stu-id="5ce54-122">In the Actual version field, enter or select a value.</span></span>
16. <span data-ttu-id="5ce54-123">Útvíkka hlutann Fjárhagstímabil fyrir dálk.</span><span class="sxs-lookup"><span data-stu-id="5ce54-123">Expand the Fiscal periods per column section.</span></span>
17. <span data-ttu-id="5ce54-124">Veljið Já í reitnum Núgildandi tímabil.</span><span class="sxs-lookup"><span data-stu-id="5ce54-124">Select Yes in the Current period field.</span></span>
18. <span data-ttu-id="5ce54-125">Útvíkka hlutann Dálkar til sýna fyrir kostnað.</span><span class="sxs-lookup"><span data-stu-id="5ce54-125">Expand the Columns to display for costs section.</span></span>
19. <span data-ttu-id="5ce54-126">Veljið Já í reitnum Fastur kostnaður.</span><span class="sxs-lookup"><span data-stu-id="5ce54-126">Select Yes in the Fixed cost field.</span></span>
20. <span data-ttu-id="5ce54-127">Veljið Já í reitnum Breytilegur kostnaður.</span><span class="sxs-lookup"><span data-stu-id="5ce54-127">Select Yes in the Variable cost field.</span></span>
21. <span data-ttu-id="5ce54-128">Veljið Já í reitnum heildarkostnaður.</span><span class="sxs-lookup"><span data-stu-id="5ce54-128">Select Yes in the Total cost field.</span></span>
22. <span data-ttu-id="5ce54-129">Smellið á „Vista“.</span><span class="sxs-lookup"><span data-stu-id="5ce54-129">Click Save.</span></span>
23. <span data-ttu-id="5ce54-130">Lokið síðunni.</span><span class="sxs-lookup"><span data-stu-id="5ce54-130">Close the page.</span></span>
24. <span data-ttu-id="5ce54-131">Fara í kostnaðarbókhald > Vinnusvæði > kostnaðarstýring.</span><span class="sxs-lookup"><span data-stu-id="5ce54-131">Go to Cost accounting > Workspaces > Cost control.</span></span>
25. <span data-ttu-id="5ce54-132">Veljið stöðu til að sjá fastan, breytilegan, heildar og óflokkaðan kostnað fyrir valin fjárhagstímabil.</span><span class="sxs-lookup"><span data-stu-id="5ce54-132">Select a statement to see fixed, variable, total, and unclassified costs for the selected fiscal periods.</span></span>
26. <span data-ttu-id="5ce54-133">Sláið inn eða veldu gildi í reitnum Fjárhagsdagatal tímabil.</span><span class="sxs-lookup"><span data-stu-id="5ce54-133">In the Fiscal calendar period field, enter or select a value.</span></span>
27. <span data-ttu-id="5ce54-134">Sláið inn eða veljið gildi í reitnum Stigveldishnútur víddar kostnaðarhlutar.</span><span class="sxs-lookup"><span data-stu-id="5ce54-134">In the Cost object dimension hierarchy node field, enter or select a value.</span></span>
    * <span data-ttu-id="5ce54-135">Eftir að þú hefur valið Stigveldi víddar fyrir kostnaðarhluta skal útvíkka Stigveldi víddar fyrir kostnaðarhluta til að sjá það kostnaðarvirði sem þú vilt.</span><span class="sxs-lookup"><span data-stu-id="5ce54-135">After you've selected a Cost object dimension hierarchy, expand the Cost element dimension hierarchy to see the desired cost values.</span></span> <span data-ttu-id="5ce54-136">Þú getur t.d. útvíkkað stigveldið í Framleiðslurekstrarkostnaður til að sjá virðið.</span><span class="sxs-lookup"><span data-stu-id="5ce54-136">For example, you can expand the hierarchy to Manufacturing overhead to see the value.</span></span>
