[!include [banner](../../includes/banner.md)]

<span data-ttu-id="f19ae-104">Eftirlitsmaður kostnaðar getur notað þetta ferli til að varpa kostnaðareiningarvídd til kostnaðareiningarvíddar í MXMF-lögaðila.</span><span class="sxs-lookup"><span data-stu-id="f19ae-104">A cost controller can use this procedure to map a cost element dimension to a cost element dimension in the MXMF legal entity.</span></span> <span data-ttu-id="f19ae-105">Þessi skráning notar USP2-sýnigagnafyrirtæki.</span><span class="sxs-lookup"><span data-stu-id="f19ae-105">This recording uses the USP2 demo data company.</span></span>

1. <span data-ttu-id="f19ae-106">Fara í kostnaðarbókhald > Víddir > Víddir kostnaðareininga.</span><span class="sxs-lookup"><span data-stu-id="f19ae-106">Go to Cost accounting > Dimensions > Cost element dimensions.</span></span>
2. <span data-ttu-id="f19ae-107">Í listanum skal finna og velja þá skráningu sem óskað er eftir.</span><span class="sxs-lookup"><span data-stu-id="f19ae-107">In the list, find and select the desired record.</span></span>
    * <span data-ttu-id="f19ae-108">Í þessu dæmi skal velja kostnaðareiningar.</span><span class="sxs-lookup"><span data-stu-id="f19ae-108">For this example, select Cost elements.</span></span>  
3. <span data-ttu-id="f19ae-109">Smellt er á Víddarvarpanir.</span><span class="sxs-lookup"><span data-stu-id="f19ae-109">Click Dimension mappings.</span></span>
4. <span data-ttu-id="f19ae-110">Smellt er á Skilgreina varpanir frá þessari vídd.</span><span class="sxs-lookup"><span data-stu-id="f19ae-110">Click Configure mappings from this dimension.</span></span>
5. <span data-ttu-id="f19ae-111">Smellið á „Nýtt“.</span><span class="sxs-lookup"><span data-stu-id="f19ae-111">Click New.</span></span>
6. <span data-ttu-id="f19ae-112">Sláið inn eða veldu gildi í reitnum í Til víddar.</span><span class="sxs-lookup"><span data-stu-id="f19ae-112">In the To dimension field, enter or select a value.</span></span>

7. <span data-ttu-id="f19ae-114">Smellið á „Nýtt“.</span><span class="sxs-lookup"><span data-stu-id="f19ae-114">Click New.</span></span>
8. <span data-ttu-id="f19ae-115">Í listanum skal merkja valda línu.</span><span class="sxs-lookup"><span data-stu-id="f19ae-115">In the list, mark the selected row.</span></span>
9. <span data-ttu-id="f19ae-116">Sláið inn eða veldu gildi í reitnum Frá víddarstak.</span><span class="sxs-lookup"><span data-stu-id="f19ae-116">In the From dimension member field, enter or select a value.</span></span>

11. <span data-ttu-id="f19ae-120">Smellið á „Vista“.</span><span class="sxs-lookup"><span data-stu-id="f19ae-120">Click Save.</span></span>
