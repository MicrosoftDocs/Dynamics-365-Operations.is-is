[!include [banner](../../includes/banner.md)]

<span data-ttu-id="99b92-104">Kostnaðarbókhald fjárhags sýnir skráningareininguna í heild.</span><span class="sxs-lookup"><span data-stu-id="99b92-104">A cost accounting ledger represents the overall reporting unit.</span></span> <span data-ttu-id="99b92-105">Það er skilgreint út frá kostnaðareiningarvídd, tölfræðilegri vídd, fjárhagsdagatali og gjaldmiðli.</span><span class="sxs-lookup"><span data-stu-id="99b92-105">It is defined by a cost element dimension, statistical dimension, fiscal calendar, and currency.</span></span> <span data-ttu-id="99b92-106">Í því felast efasemdir um hugtakið lögaðilar.</span><span class="sxs-lookup"><span data-stu-id="99b92-106">It is agnostic to the concept of legal entities.</span></span> <span data-ttu-id="99b92-107">Lögaðili og gögn hans geta tengst margvíslegu kostnaðarbókhaldi fjárhags.</span><span class="sxs-lookup"><span data-stu-id="99b92-107">A legal entity and its data can be associated with many cost accounting ledgers.</span></span> <span data-ttu-id="99b92-108">Þessi skráning notar USP2-sýnigagnafyrirtæki.</span><span class="sxs-lookup"><span data-stu-id="99b92-108">This recording uses the USP2 demo data company.</span></span>

1. <span data-ttu-id="99b92-109">Fara í Kostnaðarbókhald > Fjárhags uppsetning > Kostnaðarbókhald fjárhags.</span><span class="sxs-lookup"><span data-stu-id="99b92-109">Go to Cost accounting > Ledger setup > Cost accounting ledgers.</span></span>
2. <span data-ttu-id="99b92-110">Smellið á „Nýtt“.</span><span class="sxs-lookup"><span data-stu-id="99b92-110">Click New.</span></span>
3. <span data-ttu-id="99b92-111">Í reitinn Heiti skal slá inn gildi.</span><span class="sxs-lookup"><span data-stu-id="99b92-111">In the Name field, type a value.</span></span>
4. <span data-ttu-id="99b92-112">Sláið inn eða veljið gildi í reitnum Vídd kostnaðareiningar.</span><span class="sxs-lookup"><span data-stu-id="99b92-112">In the Cost element dimension field, enter or select a value.</span></span>
5. <span data-ttu-id="99b92-113">Sláið inn eða veldu gildi í reitnum Fjárhagsdagatal.</span><span class="sxs-lookup"><span data-stu-id="99b92-113">In the Fiscal calendar field, enter or select a value.</span></span>
6. <span data-ttu-id="99b92-114">Sláið inn eða veldu gildi í reitnum Bókhaldsgjaldmiðill.</span><span class="sxs-lookup"><span data-stu-id="99b92-114">In the Accounting currency field, enter or select a value.</span></span>
7. <span data-ttu-id="99b92-115">Sláið inn eða veldu gildi í reitnum Gerð gengis.</span><span class="sxs-lookup"><span data-stu-id="99b92-115">In the Exchange rate type field, enter or select a value.</span></span>
8. <span data-ttu-id="99b92-116">Sláið inn eða veljið gildi í reitnum Tölfræðileg vídd.</span><span class="sxs-lookup"><span data-stu-id="99b92-116">In the Statistical dimension field, enter or select a value.</span></span>
9. <span data-ttu-id="99b92-117">Smellið á „Vista“.</span><span class="sxs-lookup"><span data-stu-id="99b92-117">Click Save.</span></span>
