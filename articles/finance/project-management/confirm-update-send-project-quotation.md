[!include [banner](../includes/banner.md)]

<span data-ttu-id="e0abf-104">Þegar búið er að stofna og senda verktilboð til viðskiptavinar þarf að fá staðfestingu frá viðskiptavininum áður en staða tilboðs er uppfært í **Sent**.</span><span class="sxs-lookup"><span data-stu-id="e0abf-104">After you create and send a project quotation to a customer, you must get confirmation from the customer before you update the quotation status to **Sent**.</span></span> <span data-ttu-id="e0abf-105">Allar breytingar sem viðskiptavinur biður um er hægt að uppfæra í tilboðinu.</span><span class="sxs-lookup"><span data-stu-id="e0abf-105">Any modifications requested by the customer can be updated in the quotation.</span></span> <span data-ttu-id="e0abf-106">Þegar búið er að uppfæra stöðu á tilboðinu í **Sent** er ekki hægt að breyta henni.</span><span class="sxs-lookup"><span data-stu-id="e0abf-106">After you update the quotation status to **Sent**, you can’t modify it.</span></span> <span data-ttu-id="e0abf-107">Eftirfarandi ferli lýsir valkostum þess að senda tilboð til staðfestingar, gera uppfærslur sem byggja á ábendingum, og síðan senda tilboðið.</span><span class="sxs-lookup"><span data-stu-id="e0abf-107">The following procedure describes the options to send a quotation for confirmation, make updates based on feedback, and then send the quotation.</span></span>

## <a name="send-a-project-quotation-confirmation"></a><span data-ttu-id="e0abf-108">Senda staðfestingu verktilboðs</span><span class="sxs-lookup"><span data-stu-id="e0abf-108">Send a project quotation confirmation</span></span>  

<span data-ttu-id="e0abf-109">Hægt er að senda fyrirliggjandi verktilboð til staðfestingar í gegnum tölvupóst eða sem prentað eintak.</span><span class="sxs-lookup"><span data-stu-id="e0abf-109">You can send an existing project quotation for confirmation through email or as a printed copy.</span></span>

2. <span data-ttu-id="e0abf-111">Á síðunni **Verktilboð** skal velja tilboðið sem á að senda til staðfestingar.</span><span class="sxs-lookup"><span data-stu-id="e0abf-111">On the **Project quotation** page, select the quotation you want to send for confirmation.</span></span> 
3. <span data-ttu-id="e0abf-112">Á flipanum **Fylgja eftir**, í flokknum **Mynda**, skal velja **Staðfesta**.</span><span class="sxs-lookup"><span data-stu-id="e0abf-112">On the **Follow up** tab, in the **Generate** group, select **Confirm**.</span></span>

6. <span data-ttu-id="e0abf-116">Á síðunni **Stillingar fyrir áfangastað prentunar** skal velja **Í lagi**.</span><span class="sxs-lookup"><span data-stu-id="e0abf-116">On the **Print destination settings** page, select **OK**.</span></span>
