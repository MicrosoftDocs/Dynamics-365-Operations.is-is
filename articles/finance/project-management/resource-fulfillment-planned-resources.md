[!include [banner](../includes/banner.md)]

1. <span data-ttu-id="3f876-106">Á **Öll verk** síðunni skal velja verkið **XYZ Uppfærsla þrep 2**.</span><span class="sxs-lookup"><span data-stu-id="3f876-106">On the **All projects** page, select the **XYZ Upgrade Phase 2** project.</span></span>
2. <span data-ttu-id="3f876-107">Veljið **Verk** og svo **Breyta**.</span><span class="sxs-lookup"><span data-stu-id="3f876-107">Select **Project**, and then select **Edit**.</span></span>
3. <span data-ttu-id="3f876-108">Á flipanum **Verkhópur og áætlun** skal velja **Bæta við**.</span><span class="sxs-lookup"><span data-stu-id="3f876-108">On the **Project team and scheduling** tab, select **Add**.</span></span>
4. <span data-ttu-id="3f876-109">Í svarglugganum **Bæta við hlutverkum** veljið hlutverkið **Forritari**.</span><span class="sxs-lookup"><span data-stu-id="3f876-109">In the **Add roles** dialog box, select the **Software developer** role.</span></span>
5. <span data-ttu-id="3f876-110">Veljið **Stofna** og lokið svo verksíðunni.</span><span class="sxs-lookup"><span data-stu-id="3f876-110">Select **Create**, and then close the project page.</span></span>
6. <span data-ttu-id="3f876-111">Á síðunni **Uppfylling tilfanga** skal velja **Forritari 1** fyrir verkið **XYZ Uppfærsla verks þrep 2**.</span><span class="sxs-lookup"><span data-stu-id="3f876-111">On the **Resource fulfillment** page, select **Software developer 1** for the **XYZ Upgrade project Phase 2** project.</span></span>
7. <span data-ttu-id="3f876-112">Veljið starfskraft og svo **Úthluta**.</span><span class="sxs-lookup"><span data-stu-id="3f876-112">Select a worker, and then select **Assign**.</span></span>
8. <span data-ttu-id="3f876-113">Staðfesta að lína fyrir **Forritara 1** hefur verið fjarlægð fyrir verkið **XYZ Uppfærslu verks þrep 2**.</span><span class="sxs-lookup"><span data-stu-id="3f876-113">Verify that the line for **Software developer 1** has been removed for the **XYZ Upgrade project Phase 2** project.</span></span>
9. <span data-ttu-id="3f876-114">Á flipanum **Verkhópur Og áætlun** fyrir verkið **XYZ Uppfærsla þrep 2**, skal staðfesta að starfsmanni sem valinn var í fyrra þrepi hefur verið bætt við sem **Forritara**.</span><span class="sxs-lookup"><span data-stu-id="3f876-114">On the **Project team and scheduling** tab, for the **XYZ Upgrade Phase 2** project, verify that the worker that you selected in the previous step has been added as **Software developer**.</span></span>
