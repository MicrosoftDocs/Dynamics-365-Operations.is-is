# <a name="assign-number-sequences"></a><span data-ttu-id="ac6cd-104">Úthluta númeraröðum</span><span class="sxs-lookup"><span data-stu-id="ac6cd-104">Assign number sequences</span></span>

[!include [banner](../includes/banner.md)]
[!include [preview banner](../includes/preview-banner.md)]

<span data-ttu-id="ac6cd-105">Þetta efnisatriði útskýrir hvernig á að búa til númeraraðir fyrir kenni leigusamnings.</span><span class="sxs-lookup"><span data-stu-id="ac6cd-105">This topic explains how to create number sequences for lease IDs.</span></span> <span data-ttu-id="ac6cd-106">Það útskýrir einnig hvernig á að búa til einkvæm kenni sem eru notuð í endurmatsferli vísitölu.</span><span class="sxs-lookup"><span data-stu-id="ac6cd-106">It also explains how to create unique IDs that are used in the index revaluation process.</span></span>

1. <span data-ttu-id="ac6cd-107">Opnið **Eignarleiga \> Uppsetning \> Færibreytur fyrir eignarleigu**.</span><span class="sxs-lookup"><span data-stu-id="ac6cd-107">Go to **Asset leasing \> Setup \> Asset leasing parameters**.</span></span>
2. <span data-ttu-id="ac6cd-108">Veljið hliðarflipann **Númeraraðir**.</span><span class="sxs-lookup"><span data-stu-id="ac6cd-108">Select the **Number sequences** side tab.</span></span>
3. <span data-ttu-id="ac6cd-109">Veljið **Númeraraðir** á hliðarstikunni.</span><span class="sxs-lookup"><span data-stu-id="ac6cd-109">Select **Number sequences** in the side bar.</span></span>
4. <span data-ttu-id="ac6cd-110">Velja númeraröð fyrir tilvísun **Leigukenni**.</span><span class="sxs-lookup"><span data-stu-id="ac6cd-110">Select the number sequence for the **Lease ID** reference.</span></span> <span data-ttu-id="ac6cd-111">Þessi númeraröð verður notuð til að mynda einkvæmt kenni fyrir hvern leigusamning fyrir sig.</span><span class="sxs-lookup"><span data-stu-id="ac6cd-111">This number sequence will be used to generate the unique identifier for each lease.</span></span>
5. <span data-ttu-id="ac6cd-112">Velja númeraröð fyrir tilvísun **Vinnslukennis**.</span><span class="sxs-lookup"><span data-stu-id="ac6cd-112">Select the number sequence for the **Process ID** reference.</span></span> <span data-ttu-id="ac6cd-113">Þessi númeraröð verður notuð til að mynda einkvæmt kenni fyrir hvert endurmatsferli vísitölu.</span><span class="sxs-lookup"><span data-stu-id="ac6cd-113">This number sequence will be used to generate the unique identifier for each index revaluation process.</span></span>
6. <span data-ttu-id="ac6cd-114">Velja númeraröð fyrir tilvísun **Auðkenni riftunartillögu**.</span><span class="sxs-lookup"><span data-stu-id="ac6cd-114">Select the number sequence for the **Termination Proposal ID** reference.</span></span> <span data-ttu-id="ac6cd-115">Þessi númeraröð verður notuð til að mynda einkvæmt kenni fyrir hverja tillögu að riftun.</span><span class="sxs-lookup"><span data-stu-id="ac6cd-115">This number sequence will be used to generate the unique identifier for each termination proposal.</span></span>
