# <a name="view-journal-entries-or-transactions"></a><span data-ttu-id="cdfd7-103">Skoða færslubókarfærslur</span><span class="sxs-lookup"><span data-stu-id="cdfd7-103">View journal entries or transactions</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="cdfd7-104">Þetta ferli sýnir hvernig á að nota fyrirspurn um færslur Fylgiskjals til að leita að færslubókarfærslum eða færslum.</span><span class="sxs-lookup"><span data-stu-id="cdfd7-104">This procedure shows how to use the Voucher transactions inquiry to search for journal entries or transactions.</span></span>

1. <span data-ttu-id="cdfd7-105">Farðu í **Skoðunarrúðu > Kerfiseiningar > Fjárhagur > Fyrirspurnir og skýrslur > Færslur fylgiskjals**.</span><span class="sxs-lookup"><span data-stu-id="cdfd7-105">Go to **Navigation pane > Modules > General ledger > Inquiries and reports > Voucher transactions**.</span></span>
2. <span data-ttu-id="cdfd7-106">Veljið reit sem tilgreina á síuskilyrði fyrir.</span><span class="sxs-lookup"><span data-stu-id="cdfd7-106">Select the field for which you want to define a filter criteria.</span></span>
3. <span data-ttu-id="cdfd7-107">Færðu inn síuskilyrðið fyrir valinn reit.</span><span class="sxs-lookup"><span data-stu-id="cdfd7-107">Enter your filter critieria for the selected field.</span></span> <span data-ttu-id="cdfd7-108">Þú gætir síað á einu gildi eða sviði.</span><span class="sxs-lookup"><span data-stu-id="cdfd7-108">You could filter on a single value or a range.</span></span> <span data-ttu-id="cdfd7-109">Þegar svið er skilgreint, skal nota rétta málskipan.</span><span class="sxs-lookup"><span data-stu-id="cdfd7-109">When defining a range, make sure the correct syntax is used.</span></span> <span data-ttu-id="cdfd7-110">Gildin skal aðskilja með tvöföldum punkti (..).</span><span class="sxs-lookup"><span data-stu-id="cdfd7-110">The values should be separated by a double period (..).</span></span>  
4. <span data-ttu-id="cdfd7-111">Smellið á flipann **Samtengingar** til að bæta við viðbótartöflum sem á að sía úr.</span><span class="sxs-lookup"><span data-stu-id="cdfd7-111">Click the **Joins** tab to add additional tables from which to filter.</span></span>
5. <span data-ttu-id="cdfd7-112">Í trénu skal velja **Töflur/Færsla í almenna færslubók**.</span><span class="sxs-lookup"><span data-stu-id="cdfd7-112">In the tree, select **Tables/General journal entry**.</span></span>
6. <span data-ttu-id="cdfd7-113">Smelltu á **Bæta við töflusamtengingu**.</span><span class="sxs-lookup"><span data-stu-id="cdfd7-113">Click **Add table join**.</span></span>
7. <span data-ttu-id="cdfd7-114">Smelltu á **Hætta við** ef ákveðið er að bæta ekki við aukatöflu.</span><span class="sxs-lookup"><span data-stu-id="cdfd7-114">Click **Cancel** if you decide not to add an additional table.</span></span>
8. <span data-ttu-id="cdfd7-115">Smelltu á flipann **Svið**.</span><span class="sxs-lookup"><span data-stu-id="cdfd7-115">Click the **Range** tab.</span></span>
9. <span data-ttu-id="cdfd7-116">Smellið á **Í lagi** til að keyra fyrirspurn.</span><span class="sxs-lookup"><span data-stu-id="cdfd7-116">Click **OK** to run the query.</span></span>
10. <span data-ttu-id="cdfd7-117">Smelltu á **Færsluuppruni** á aðgerðarúðunni.</span><span class="sxs-lookup"><span data-stu-id="cdfd7-117">On the Action pane, click **Transaction origin**.</span></span> <span data-ttu-id="cdfd7-118">Ýmsir hnappar í kringum hnitanetið má nota til að rannsaka viðbótarupplýsingar um valda færslu fylgiskjalsins.</span><span class="sxs-lookup"><span data-stu-id="cdfd7-118">Various buttons about the grid can be used to research additional information about the selected record of the voucher.</span></span> <span data-ttu-id="cdfd7-119">Sumir hnappar eru hugsanlega ekki tiltækir, það fer eftir gerð færslu og eiginleikum færslu.</span><span class="sxs-lookup"><span data-stu-id="cdfd7-119">Some buttons may not be available, depending on the type of transaction and characteristics of the transaction.</span></span>
11. <span data-ttu-id="cdfd7-120">Lokið síðunni.</span><span class="sxs-lookup"><span data-stu-id="cdfd7-120">Close the page.</span></span>
12. <span data-ttu-id="cdfd7-121">Smellið á **Upphaflegt skjal** á aðgerðarúðunni.</span><span class="sxs-lookup"><span data-stu-id="cdfd7-121">On the Action pane, Click **Original document**.</span></span>
13. <span data-ttu-id="cdfd7-122">Lokið síðunni.</span><span class="sxs-lookup"><span data-stu-id="cdfd7-122">Close the page.</span></span>
