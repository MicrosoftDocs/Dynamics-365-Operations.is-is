# <a name="create-consolidation-groups-and-additional-consolidation-accounts"></a>Stofna samstæðuhópa og fleiri samstæðulykla

[!include [banner](../../includes/banner.md)]

Þessi verklýsing sýnir hvernig stofna á flokk samstæðulykla og bæta síðan lykla í flokkinn. Þessi aðferð notar sýnifyrirtækið USMF.

## <a name="create-a-consolidation-account-group"></a>Flokkur samstæðulykla stofnaður
1. Fara til **Fjárhagur > Reikningsyfirlit > Reikningar > Samstæðureikningaflokkar**.
2. Smellt er á **Nýtt**.
3. Í **Samstæðureikningahópur** reit, sláðu inn einstakt auðkenni fyrir samstæðureikningahópinn.
4. Í reitinn **Heiti** skal slá inn gildi.

## <a name="add-accounts-to-consolidation-account-group"></a>Bæta lyklum við flokk samstæðulykla
1. Fara til **Fjárhagur > Reikningsyfirlit > Reikningar > Viðbótarsamstæðureikningar**.
2. Smellt er á **Nýtt**.
3. Í **Aðalreikningur** veldu á listanum aðalreikninginn sem þú vilt kortleggja.
4. Í **Samstæðureikningahópur** reit, á listanum, smelltu á samstæðureikningahópinn.
5. Í **Samstæðureikningur** reit, sláðu inn gildi.
6. Í **Heiti samstæðureiknings** reit, sláðu inn gildi.
