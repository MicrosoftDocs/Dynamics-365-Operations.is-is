# <a name="maintain-financial-dimension-default-templates"></a><span data-ttu-id="2f026-103">Viðhalda sjálfgefnum sniðmátum fjárhagsvídda</span><span class="sxs-lookup"><span data-stu-id="2f026-103">Maintain financial dimension default templates</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="2f026-104">Þetta efni sýnir hvernig á að setja upp sjálfgefin sniðmát fjárhagsvídda.</span><span class="sxs-lookup"><span data-stu-id="2f026-104">This topic shows how to set up financial dimension default templates.</span></span> <span data-ttu-id="2f026-105">Hægt er að nota sniðmát til að dreifa upphæðum á upprunaskjöl á fljótlegan hátt.</span><span class="sxs-lookup"><span data-stu-id="2f026-105">The templates can be used to quickly distribute amounts on source documents.</span></span>

1. <span data-ttu-id="2f026-106">Í skoðunarrúðunni ferðu í **Kerfiseiningar > Fjárhag > Bókhaldslykill > Víddir > Sjálfgefin sniðmát fjárhagsvídda**.</span><span class="sxs-lookup"><span data-stu-id="2f026-106">In the navigation pane, go to **Modules > General ledger > Chart of accounts > Dimensions > Financial dimension default templates**.</span></span>
2. <span data-ttu-id="2f026-107">Veljið **Nýtt**.</span><span class="sxs-lookup"><span data-stu-id="2f026-107">Select **New**.</span></span>
3. <span data-ttu-id="2f026-108">Í reitinn **Sniðmátskenni** skal færa inn sniðmátskenni.</span><span class="sxs-lookup"><span data-stu-id="2f026-108">In the **Template ID** field, enter a Template ID.</span></span>
4. <span data-ttu-id="2f026-109">Færa inn prósentu fyrir fyrstu víddargildin.</span><span class="sxs-lookup"><span data-stu-id="2f026-109">Enter the percentage for the first dimension values.</span></span>
5. <span data-ttu-id="2f026-110">Færa inn fjárhagsvíddargildi sem samsvara innfærðu hlutfalli.</span><span class="sxs-lookup"><span data-stu-id="2f026-110">Enter the financial dimension values that correspond to the percent entered.</span></span>
6. <span data-ttu-id="2f026-111">Veljið **Bæta við**.</span><span class="sxs-lookup"><span data-stu-id="2f026-111">Select **Add**.</span></span>
7. <span data-ttu-id="2f026-112">Færa inn aðra prósentu fyrir næstu víddargildi.</span><span class="sxs-lookup"><span data-stu-id="2f026-112">Enter another percent for the next dimension values.</span></span> <span data-ttu-id="2f026-113">Í þessu dæmi skal færa inn prósentu svo þú sért enn með lægra en 100% úthlutað.</span><span class="sxs-lookup"><span data-stu-id="2f026-113">For this example, enter a percentage so you still have less than 100% allocated.</span></span> 
8. <span data-ttu-id="2f026-114">Færa inn fjárhagsvíddargildi sem samsvara innfærðu hlutfalli.</span><span class="sxs-lookup"><span data-stu-id="2f026-114">Enter the financial dimension values that correspond to the percent entered.</span></span>
9. <span data-ttu-id="2f026-115">Veljið **Bæta við**.</span><span class="sxs-lookup"><span data-stu-id="2f026-115">Select **Add**.</span></span>
10. <span data-ttu-id="2f026-116">Færið inn aðra prósentu sem munu gera heildarprósenta jafnt og 100%.</span><span class="sxs-lookup"><span data-stu-id="2f026-116">Enter another percent, which will make the total percent equal to 100%.</span></span>
11. <span data-ttu-id="2f026-117">Færa inn fjárhagsvíddargildi sem samsvara innfærðu hlutfalli.</span><span class="sxs-lookup"><span data-stu-id="2f026-117">Enter the financial dimension values that correspond to the percent entered.</span></span>
12. <span data-ttu-id="2f026-118">Veljið **Vista**.</span><span class="sxs-lookup"><span data-stu-id="2f026-118">Select **Save**.</span></span>
