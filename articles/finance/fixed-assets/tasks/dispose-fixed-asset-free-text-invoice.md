# <a name="dispose-of-a-fixed-asset-using-a-free-text-invoice"></a><span data-ttu-id="db4b9-103">Selja eign með reikningi með frjálsum texta</span><span class="sxs-lookup"><span data-stu-id="db4b9-103">Dispose of a fixed asset using a free text invoice</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="db4b9-104">Þetta efni útskýrir hvernig losnað er við eign með reikningi með frjálsum texta.</span><span class="sxs-lookup"><span data-stu-id="db4b9-104">This topic explains how to dispose of a fixed asset using the free text invoice.</span></span>

1. <span data-ttu-id="db4b9-105">Í skoðunarrúðunni ferðu í **Einingar > Viðskiptakröfur > Reikningar > Allir frjálsir textareikningar**.</span><span class="sxs-lookup"><span data-stu-id="db4b9-105">In the navigation pane, go to **Modules > Accounts receivable > Invoices > All free text invoices**.</span></span>
2. <span data-ttu-id="db4b9-106">Veljið **Nýtt**.</span><span class="sxs-lookup"><span data-stu-id="db4b9-106">Select **New**.</span></span>
3. <span data-ttu-id="db4b9-107">Í reitnum **Viðskiptavinalykill** skal færa inn eða velja gildi.</span><span class="sxs-lookup"><span data-stu-id="db4b9-107">In the **Customer account** field, enter or select a value.</span></span>
4. <span data-ttu-id="db4b9-108">Staðfesta sjálfgefna dagsetningu **Reiknings** og breyta ef við á.</span><span class="sxs-lookup"><span data-stu-id="db4b9-108">Validate the default **Invoice** date and edit if applicable.</span></span>
5. <span data-ttu-id="db4b9-109">Staðfesta útistandandi sjálfgefna hausareiti á borð við **Gjaldmiðil** og breytið ef við á.</span><span class="sxs-lookup"><span data-stu-id="db4b9-109">Validate remaining default header fields, such as **Currency** and edit if applicable.</span></span>
6. <span data-ttu-id="db4b9-110">Í kaflanum **Reikningslínur** skal staðfesta reitina **Lýsing** og **Aðallykill**.</span><span class="sxs-lookup"><span data-stu-id="db4b9-110">In the **Invoice lines** section, validate the **Description** and **Main account** fields.</span></span>
7. <span data-ttu-id="db4b9-111">Staðfesta sjálfgefnu reitina **VSK-flokka** og **VSK-flokka fyrir vörur**.</span><span class="sxs-lookup"><span data-stu-id="db4b9-111">Validate the default **Sales tax group** and **Item sales tax group** fields.</span></span>
8. <span data-ttu-id="db4b9-112">Sláðu inn **Einingarverð** eða **Upphæð** fyrir sölu eignarinnar.</span><span class="sxs-lookup"><span data-stu-id="db4b9-112">Enter the **Unit price** or the **Amount** of the sale of the fixed asset.</span></span>
9. <span data-ttu-id="db4b9-113">Veldu hlutann **Línuupplýsingar**.</span><span class="sxs-lookup"><span data-stu-id="db4b9-113">Select the **Line details** section.</span></span>  
10. <span data-ttu-id="db4b9-114">Í reitnum **Eignaflokkur** skal færa inn eða velja gildi.</span><span class="sxs-lookup"><span data-stu-id="db4b9-114">Enter or select a value in the **Fixed asset** field.</span></span>
11. <span data-ttu-id="db4b9-115">Veldu **Bóka**.</span><span class="sxs-lookup"><span data-stu-id="db4b9-115">Select **Post**.</span></span>
