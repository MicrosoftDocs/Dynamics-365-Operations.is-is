# <a name="set-up-and-create-depreciation-profiles"></a>Setja upp og búa til afskriftarreglur

[!include [banner](../../includes/banner.md)]

Afskriftareglur ákvarða tegund og tíðni afskriftar fyrir eigna.


## <a name="create-a-depreciation-profile"></a>Stofna afskriftarreglu
1. Í skoðunarrúðnni ferðu í **Kerfseiningar > Fastafjármunir > Uppsetning > Afskriftareglur**.
2. Smellt er á **Nýtt**.
3. Í reitnum **Afskriftareglur** skaltu slá inn gildi.
4. Í reitinn **Heiti** skal slá inn gildi.
5. Veljið valkost í retinum **aðferð**. Ef valið er lækka stöðu þarf að færa inn hlutfall í reitinn **Prósenta**.  
6. Í reitnum **Afskriftarár** skal velja valkost.
7. Í reitnum **Tímabilstíðni** skal velja valkost.
8. Lokaðu skjámyndinni.
