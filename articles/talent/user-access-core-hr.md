# <a name="user-can-access-core-hr-but-not-the-onboard-or-attract-app"></a>Notandi getur opnað Core HR en ekki forritin Onboard eða Attract

[!include [banner](includes/banner.md)]

**Umhverfisupplýsingar**

- Notandi A sá um uppsetningu Microsoft Dynamics Lifecycle Services (LCS).
- Notandi A bætti notanda B við sem notanda í Microsoft Dynamics 365 for Talent Core HR.

**Úthreyfing**

Notandi B getur opnað Core HR, en hann getur ekki opnað forritið Talent: Attract eða Talent: Onboard. Þegar notandi reynir að opna **Forrit fyrir umhverfi** er honum vísað í prófunarumhverfi í staðinn.

**Lausn**

Nauðsynlegt er að úthluta réttindum á Notanda B til að skoða umhverfi Microsoft PowerApps sem notandi A stofnaði í úthlutunarferlinu.

Upplýsingar er að finna í „Að veita aðgang að umhverfinu“ í kaflanum [Úthlutun Talent](https://docs.microsoft.com/dynamics365/unified-operations/talent/provisioning-talent).

**Langtímalausn**

Microsoft er að íhuga að úthluta sjálfkrafa viðeigandi réttindum fyrir Onboard og Attract þegar notanda er bætt við Core HR.
