# <a name="whats-new-or-changed-in-dynamics-365-for-talent-july-02-2019"></a>Nýjungar eða breytingar í Dynamics 365 for Talent (2. júlí 2019)

[!include [banner](includes/banner.md)]

Í þessu efnisatriði er að finna lýsingu á nýjum eða breyttum eiginleikum í Microsoft Dynamics 365 for Talent.

## <a name="changes-in-attract"></a>Breytingar í Attract

### <a name="use-organization-service-accounts-to-send-e-mails"></a>Notaðu þjónustureikninga fyrirtækisins til að senda tölvupóst

Þú getur nú stillt tölvupóststillingar þannig að tölvupóstur til umsækjenda, viðmælenda, ráðgjafar og ráðningarstjórar sé sendur úr póstþjónustu fyrirtækisins. Nánari upplýsingar er að finna í [Tengja tölvupóstþjónustureikning](./attract-configure-email-settings.md#connect-an-email-service-account).

#### <a name="offer-email-templates"></a>Tilboðssniðmát fyrir tölvupóst 

Þú getur nú stillt tölvupóst sniðmát stofnunarinnar fyrir öll stig meðan á undirbúningi, samþykki og staðfestingu stendur. Notendur geta annaðhvort notað sjálfgefið sniðmát eða sérsniðið það áður en þú sendir tölvupóst. Nánari upplýsingar er að finna í [Skilgreina sniðmátsstillingar tölvupósts](./attract-configure-email-settings.md#configure-email-template-settings).

### <a name="candidate-consent-on-application"></a>Samþykkt umsækjanda í umsókn

Þú getur nú virkjað tengilinn Skilmála fyrir fyrirtækið þitt. Með því að virkja þessa stillingu þurfa umsækjendur að samþykkja skilmála áður en þeir leggja inn starfsumsóknir sínar. 

## <a name="changes-in-onboard"></a>Breytingar í Onboard

Þessi útgáfa inniheldur minniháttar villuleiðréttingar fyrir Dynamics 365 Talent: Onboard.

## <a name="changes-in-core-hr"></a>Breytingar í Core HR
