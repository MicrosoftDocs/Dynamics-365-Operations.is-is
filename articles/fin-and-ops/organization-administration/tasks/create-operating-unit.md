[!include[task guide banner](../../includes/task-guide-banner.md)]

    * <span data-ttu-id="761fd-112">Veldu þá tegund rekstrareiningar sem á að stofna.</span><span class="sxs-lookup"><span data-stu-id="761fd-112">Select the type of operating unit you want to create.</span></span>  
4. <span data-ttu-id="761fd-113">Í listanum skal smella á tengilinn í valinni línu.</span><span class="sxs-lookup"><span data-stu-id="761fd-113">In the list, click the link in the selected row.</span></span>

    * <span data-ttu-id="761fd-115">Útvíkka hlutann Almennt ef þarf.</span><span class="sxs-lookup"><span data-stu-id="761fd-115">Expand the General section, if necessary.</span></span>
