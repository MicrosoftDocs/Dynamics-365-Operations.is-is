# <a name="job-detail-api"></a>Forritaskil upplýsinga um starf

Þessi grein veitir upplýsingar og dæmi um fyrirspurn fyrir eininguna um starfsupplýsingar í Dynamics 365 Human Resources.

## <a name="properties"></a>Eiginleikar

| Eiginleiki<br>**Efnislegt heiti**<br>**_Gerð_** | Nota | lýsing |
| --- | --- | --- |
| **Starfsauðkenni**<br>mshr_jobid<br>*Strengur* | Lesa eingöngu<br>Krafa | Einkvæmt kenni fyrir verk. |
| **Neðri þröskuldur markaðsverðs**<br>mshr_marketpricelowthreshold<br>*Tugabrot* | | GUID-gildi myndað af kerfinu til að auðkenna stöðu á einkvæman hátt.  |
