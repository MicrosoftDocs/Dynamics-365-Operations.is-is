# <a name="marital-status"></a>Hjúskaparstaða

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Þetta efnisatriði lýsir valmöguleika hjúskaparstöðu fyrir Dynamics 365 Human Resources.

Raunlægt heiti: mshr_dirpersonmaritalstatus

Upptalningin býður upp á valmöguleika fyrir hjúskaparstöðu í einstaklingsfærslum.

| Virði | Merkimiði | lýsing |
| --- | --- | --- |
| 200000000 | Engum | Ekkert gildi hefur verið valið.
| 200000001 | Einhleyp(ur) | Stakt. |
| 200000002 | Gift(ur) | Gift(ur). |
| 200000003 | Fráskilin(n) | Fráskilin(n). |
| 200000004 | Ekkja/ekkill | Ekkja/Ekkill. |
