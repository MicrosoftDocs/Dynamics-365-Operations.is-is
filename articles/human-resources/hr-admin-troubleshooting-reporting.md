# <a name="reporting-options"></a>Valkostir tilkynninga


[!INCLUDE [PEAP](../includes/peap-2.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]



**Umhverfisupplýsingar**

Þetta vandamál á við um öll umhverfi.

**Einkenni**

Viðskiptamaður vill sérsníða skýrslur eða stofna nýjar Microsoft Dynamics 365 Human Resources skýrslur.

**Úthreyfing**

Notandi getur ekki sérsniðið innfelldar Microsoft Power BI skýrslur.

**Lausn**

- Hægt er að gefa skýrslu um gögn Human Resources sem flæða til Dataverse í gegnum Power Apps Dataverse-tengingu við Power BI Desktop. Athugið að Dataverse inniheldur undirsafn af Human Resources-gögnum. Frekari upplýsingar um Power BI og yfirlit eru í [Stofna Power BI-skýrslur og yfirlit með Power Apps Common Data Service](https://powerapps.microsoft.com/blog/cdsconnectortopowerbi).
- Rafræn skýrslugerð (ER) er í boði fyrir sumar skýrslur í Human Resources. Hægt er að gera sérstillingar, knúnar af viðskiptamönnum, með stillingarmöguleikum rafrænnar skýrslugerðar.
- Hægt er að flytja út gögn í Microsoft Excel eða Microsoft Word með ýmsum gagnaeiningum sem Human Resources útvegar í gegnum samþættingu Microsoft Office.

**Langtímalausn**

Frekari Power BI-valkostir verða tiltækir og fleiri gögn og einingar verða hluti af Dataverse.
