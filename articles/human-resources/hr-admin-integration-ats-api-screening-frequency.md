# <a name="screening-frequency"></a>Tíðni skoðunar

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Þetta efnisatriði lýsir valmöguleika skoðunartíðni fyrir Dynamics 365 Human Resources.

Raunlægt heiti: mshr_hcmfrequencyunit

Þessi upptalning býður upp á gildi valmöguleika fyrir tíðni skoðunar. 

| Virði | Merkimiði | lýsing |
| --- | --- | --- |
| 200000000 Aðeins einu sinni | Skoðunar er aðeins krafist einu sinni. |
| 200000001 Daglega | Tíðni skoðunar er reiknuð í dögum. |
| 200000002 Vikulega | Tíðni skoðunar er reiknuð í vikum. |
| 200000003 Mánaðarlega | Tíðni skoðunar er reiknuð í mánuðum. |
| 200000004 Árlega | Tíðni skoðunar er reiknuð í árum. |
