# <a name="skill-level-type"></a>Gerð hæfnisstig


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Þessi grein lýsir valmöguleikanum fyrir hæfnistigstegund fyrir Dynamics 365 Human Resources.

Raunlægt heiti: mshr_hrmskillleveltype

Þessi upptalning býður upp á gildi valmöguleika sem flokka stigið sem úthlutað er á hæfni umsækjanda.

| Virði | Merkimiði | lýsing |
| --- | --- | --- |
| 200000000 | Raunþyngd | Raunverulegt hæfnisstig. |
| 200000001 | Mark | Markgildi fyrir hæfnisstigið. |
