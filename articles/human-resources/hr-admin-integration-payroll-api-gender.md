# <a name="payroll---gender-option-set"></a>Launaskrá - Valmöguleikar kyns


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Þetta efnisatriði lýsir valmöguleika kyns fyrir Dynamics 365 Human Resources.

## <a name="option-set-gender-option"></a>Valmöguleiki: Valkostur kyns

Þessi upptalning býður upp á valmöguleika kyns fyrir starfsmanninn. Þetta er tiltækt á mshr_hcmpersongender option set.

| Virði | Merkimiði | lýsing |
| --- | --- | --- |
| 200000000 | Engum | Ekkert kyn hefur verið tilgreint. |
| 200000001 | Karl | Karl. |
| 200000002 | Kona | Kona. |
| 200000003 | Ótilgreint | Valkostur fyrir ótilgreint kyn. |
