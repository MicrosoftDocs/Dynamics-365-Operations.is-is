# <a name="regulatory-job-category"></a>Starfsflokkur samkvæmt reglum

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Þetta efnisatriði lýsir valmöguleika fyrir starfsflokk samkvæmt reglum fyrir Dynamics 365 Human Resources.

Raunlægt heiti: mshr_hcmregulatoryjobcategory

Þessi upptalning tilgreinir valmöguleika fyrir gildi starfsflokks samkvæmt reglum. Boðið er upp á þetta í fyrirliggjandi valmöguleika mshr_hcmregulatoryjobcategory.

| Virði | Merkimiði | lýsing |
| --- | --- | --- |
| 200000000 | Engum | Ekkert. |
| 200000001 | Yfirmaður | Framkvæmdastjórn/æðsta stig embættismanna og stjórnenda. |
| 200000002 | Stjórnandi | Opinberir starfsmenn og stjórnendur á æðsta stigi/miðstigi. |
| 200000003 | Fræðilegur | Sérfræðingar. |
| 200000004 | Tæknimaður | Tæknimenn. |
| 200000005 | SalesWorker | Starfsmenn sölu. |
| 200000006 | Stjórnun | Starfsmenn stjórnendaþjónustu. |
| 200000007 | CraftWorker | Handverksstarfsmenn. |
| 200000008 | Rekstraraðili | Rekstraraðilar. |
| 200000009 | Verkamaður | Verkamenn/aðstoðarfólk. |
| 200000010 | ServiceWorker | Þjónustuaðilar. |
