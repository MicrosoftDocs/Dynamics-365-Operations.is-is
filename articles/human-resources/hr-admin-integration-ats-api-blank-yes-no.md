# <a name="blank-yes-no"></a>Autt Já Nei


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Þetta efnisatriði lýsir auða Já Nei valkostinum fyrir Dynamics 365 Human Resources.

Raunlægt heiti: mshr_hrmblankyesno

Þessi tölusetning býður upp á valkosti fyrir já/nei eiginleika sem leyfa einnig auða reiti.

| Virði | Merkimiði | lýsing |
| --- | --- | --- |
| 200000000 | Autt | Ekkert gildi hefur verið valið. |
| 200000001 | Já | Já. |
| 200000002 | Nei | Nei. |
