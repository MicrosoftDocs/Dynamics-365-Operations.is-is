# <a name="education-credit-basis"></a>Einingagrunnur menntunar

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Þetta efnisatriði lýsir valmöguleikum einingagrunns menntunar fyrir Dynamics 365 Human Resources.

Raunlægt heiti: mshr_hcmeducationcreditbasis

Þessi upptalning býður upp á valkosti gildi fyrir einingagrunn menntunar í menntunarfærslu umsækjanda.

| Virði | Merkimiði | lýsing |
| --- | --- | --- |
| 200000000 | Autt | Ekkert gildi hefur verið valið. |
| 200000001 | Misseri | Misseri. |
| 200000002 | Ársfjórðungur | Ársfjórðungur. |
| 200000003 | Þriðjungsönn | Þriðjungsönn. |
| 200000004 | Hugtak | Hugtak. |
| 200000005 | Annað | Annað. |
