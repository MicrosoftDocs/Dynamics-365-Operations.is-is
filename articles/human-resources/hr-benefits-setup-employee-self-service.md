# <a name="configure-employee-self-service"></a>Grunnstilla sjálfsafgreiðslu starfsmanns

[!include [banner](includes/preview-feature.md)]

Í Microsoft Dynamics 365 Human Resources, þú getur stillt flísar fyrir toppstig siglingar í sjálfsafgreiðslu starfsmanna. Ávinningur áætlunarflísar beinir notendum að hagnaði áætlunum sem þeir eru gjaldgengir í.

## <a name="set-up-a-role-center-tile"></a>Uppsetning á reit hlutverkamiðstöðvar

1. Í vinnusvæðinu **Fríðindastjórnun**, undir **Skipulag**, veldu **sjálfsafgreiðslufæribreytur starfsmanns**.

2. Veldu **Skipulag hlutverks miðflísar** flipanum og veldu síðan **Nýtt**.

3. Tilgreina gildi fyrir eftirfarandi reiti:

   | Svæði | Lýsing |
   | --- | --- |
   | Auðkenni reits | Einkvæmt auðkenni fyrir reitinn. |
   | Texti merkis reits | Textinn sem birtist fyrir reitinn í sjálfsafgreiðslunni. |
   | Lýsing | Lýsing á rietnum. |
   | Internet-fang | Sláðu slóðina inn á sjálfsafgreiðslusíðuna fyrir starfsmenn. |
   | Stærð reits | Stærð flísanna: Lítil, meðalstór eða stór. |
   | Mark | Tilgreinir hvort síðan eigi að opna í nýjum glugga eða núverandi glugga. |
   | Bakgrunnsmynd reits | Slóðin á myndina sem á að nota fyrir reitinn (valfrjálst). |
   | Hefst | Upphafsdagsetning og tími sem reiturinn ætti að vera tiltækur. |
   | Lýkur | Lokasetning og tími sem reiturinn ætti að vera tiltækur. |

4. Veljið **Vista**.
