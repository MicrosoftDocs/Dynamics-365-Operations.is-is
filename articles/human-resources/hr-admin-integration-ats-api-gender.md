# <a name="gender"></a>Kyn

Þetta efnisatriði lýsir valmöguleika kyns fyrir Dynamics 365 Human Resources.

Raunlægt heiti: mshr_hcmpersongender

Þessi upptalning býður upp á valmöguleika kyns fyrir umsækjandann. Þetta er tiltækt á mshr_hcmpersongender option set.

| Virði | Merkimiði | lýsing |
| --- | --- | --- |
| 200000000 | Engum | Ekkert kyn hefur verið tilgreint. |
| 200000001 | Karl | Karl. |
| 200000002 | Kona | Kona. |
| 200000003 | Ótilgreint | Valkostur fyrir ótilgreint kyn. |
