# <a name="set-up-tier-codes"></a>Setja upp kóða fyrir lög

Dynamics 365 Human Resources notar lagaaupplýsingar með fríðindataxta sem eru greiddir af starfsmönnum og vinnuveitanda. Þú getur skilgreint ótakmarkaðan fjölda flokkana og sett upp útreikninga á gengi til að mæta þörfum fyrirtækisins.

1. Í vinnusvæðinu **Fríðindastjórnun**, undir **Skipulag**, veldu **Lagakóða**.

2. Veljið **Nýtt**.

3. Tilgreina gildi fyrir eftirfarandi reiti:

   | Svæði | Lýsing |
   | --- | --- |
   | **Kóði lags** | Sérstakur fríðindalagakóði eða -númer til að bera kennsl á laganúmerið. |
   | **Lýsing** | Lýsing á fríðindalagakóða. |

4. Veljið **Vista**.
