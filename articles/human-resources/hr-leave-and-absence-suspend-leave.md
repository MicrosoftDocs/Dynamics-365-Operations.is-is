[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

<span data-ttu-id="cc213-104">Þú getur frestað leyfi starfsmanns til að stöðva vinnslur á leyfisuppsöfnunum vegna valinna leyfisgerða.</span><span class="sxs-lookup"><span data-stu-id="cc213-104">You can suspend leave for an employee to stop leave accruals from being processed for selected leave types.</span></span>

1. <span data-ttu-id="cc213-106">Veldu á skrá starfsmannsins **Leyfi**.</span><span class="sxs-lookup"><span data-stu-id="cc213-106">On the employee's record, select **Leave**.</span></span>

2. <span data-ttu-id="cc213-107">Veldu **Fresta leyfi**.</span><span class="sxs-lookup"><span data-stu-id="cc213-107">Select **Suspend leave**.</span></span>

3. <span data-ttu-id="cc213-108">Veljið **Nýtt**.</span><span class="sxs-lookup"><span data-stu-id="cc213-108">Select **New**.</span></span>

4. <span data-ttu-id="cc213-109">Í valmyndinni **Fresta leyfisuppsöfnun** velurðu **Gerð leyfis** ásamt **Upphafsdagur** og **Lokadagur** vegna frestunarinnar.</span><span class="sxs-lookup"><span data-stu-id="cc213-109">In the **Suspend leave accrual** dialog box, select the **Leave type** along with the **Start date** and **End date** for the suspension.</span></span>

<span data-ttu-id="cc213-111">Sé unnið úr uppsöfnunum á meðan leyfi starfsmanns er frestað verður engin uppsöfnun gerð fyrir frestaðar leyfisgerðir.</span><span class="sxs-lookup"><span data-stu-id="cc213-111">If accruals are processed while the employee's leave is suspended, no accrual will be made for the suspended leave types.</span></span>
