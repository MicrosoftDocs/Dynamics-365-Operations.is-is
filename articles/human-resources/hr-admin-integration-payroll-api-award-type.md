# <a name="award-type"></a>Tegund umbunar

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Þetta efnisatriði lýsir valmöguleikum fyrir tegund umbunar fyrir Dynamics 365 Human Resources.

## <a name="option-set-award-type"></a>Valmöguleikar: Tegund umbunar

Efnislegt heiti: mshr_HrmCompVarAwardEmplType

Þessi upptalning gefur kost á að valmöguleikunum fyrir tegund umbunar fyrir breytilega uppbót í sýndareiningum Human Resources.

| Virði | Merkimiði | lýsing |
| --- | --- | --- |
| 200000000 | Almenn umbun | Almenn umbun. |
| 200000001 | Heildarsamtala fastra launa | Heildarsamtala fastra launa. |
