# <a name="removed-or-deprecated-features-in-dynamics-365-human-resources"></a>Fjarlægðir eða úreltir eiginleikar í Dynamics 365 Human Resources

Þessi grein lýsir eiginleikum sem hafa verið fjarlægðir, eða sem fyrirhugað er að fjarlægja úr Microsoft Dynamics 365 Mannauður.

- *Fjarlægður* eiginleiki er ekki lengur tiltækur í vörunni.
- *Úreltur* eiginleiki er ekki í virkri þróun og kann að vera fjarlægður úr uppfærslum í framtíðinni.

Þessi listi er ætlað að hjálpa þér að íhuga þessar fjarlægingar og úreldingar fyrir eigin áætlanagerð.

## <a name="features-removed-or-deprecated-in-human-resources"></a>Eiginleikar fjarlægðir eða úreltir í Mannauður 

### <a name="new-lcs-project-creation-in-human-resources-stand-alone"></a>Ný LCS verkefnasköpun í mannauði (sjálfstætt) 

[Úthlutun Mannauður](/hr-admin-setup-provision)

| &nbsp; | &nbsp; |
|------------|--------------------|
| **Ástæða úreldingar/fjarlægingar** | Skipt út fyrir verkefnasköpun á fjármála- og rekstrarinnviðum. Til að læra meira um samruna innviða, sjá [Algengar spurningar um sameiningu mannauðsinnviða Dynamics 365](/hr-infrastructure-merge-faq). |
| **Skipt út fyrir aðra eiginleika?**   | Já |
| **Afurðasvæði sem haft er áhrif á**         | Forrit |
| **Dreifingarvalkostur**              | Allir |
| **Staða**                         | Úrelt: Frá og með júlí 2022 styðjum við ekki lengur að búa til ný verkefni í mannauði (sjálfstætt). |
