# <a name="no-yes"></a>Nei Já

Þetta efnisatriði lýsir valmöguleika Nei Já fyrir Dynamics 365 Human Resources.

## <a name="option-set-no-yes"></a>Valmöguleiki: Nei Já

Raunlægt heiti: mshr_noyes

Þessi upptalning býður upp á valmöguleika fyrir dæmigerða Boole-eiginleika í sýndareiningum Human Resources.

| Virði | Merkimiði | lýsing |
| --- | --- | --- |
| 200000000 | Ekkert | Nei. |
| 200000001 | Já | Já. |
