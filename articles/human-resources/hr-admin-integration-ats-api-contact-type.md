# <a name="contact-type"></a>Gerð tengiliðar

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Þessi upptalning býður upp á valmöguleika yfir gildi fyrir tengiliðagerðir umsækjanda. 

| Virði | Merkimiði | lýsing |
| --- | --- | --- |
| 200000000 | Engum | Engin gerð er valin. |
| 200000001 | Sími | Símanúmer. |
| 200000002 | Tölvupóstfang | Netfang. |
| 200000003 | URL | Vefslóð vefsvæðis. |
| 200000004 | Telex | Telexnúmer. |
| 200000005 | Fax | Faxnúmer. |
| 200000006 | Facebook | Facebook reikningur. Auðkennt af notandakenni. |
| 200000007 | Twitter | Twitter-reikningur. Auðkennt með @username. |
| 200000008 | LinkedIn | LinkedIn-reikningur. Auðkennt með notandanafni. |
