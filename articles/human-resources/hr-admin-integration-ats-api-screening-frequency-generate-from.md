# <a name="screening-frequency-generate-from"></a>Tíðni skoðunar mynduð úr


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Þessi grein lýsir skimunartíðninni sem myndast úr valkostinum sem er stilltur fyrir Dynamics 365 Human Resources.

Raunlægt heiti: mshr_hcmfrequencygeneratefrom

Þessi upptalning býður upp á gildi valmöguleika til að ákveða upphafsdagsetningu útreiknings fyrir næstu áskildu skoðun.

| Virði | Merkimiði | lýsing |
| --- | --- | --- |
| 200000000 Ekki valið | Ekkert gildi hefur verið valið. |
| 200000001 Dagsetning þegar lokið | Útreikningur er gerður á síðasta lokna skoðunardegi. |
| 200000002 Dagsetning áskilin | Útreikningur er gerður á síðasta áskilda skoðunardegi. |
