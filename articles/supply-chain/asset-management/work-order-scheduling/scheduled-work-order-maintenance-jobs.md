<span data-ttu-id="dc2ab-104">Síðan **Skipulögð viðhaldsverk verkbeiðna** er sýnir yfirlit yfir verkbeiðnir sem úthlutað er á tilföng.</span><span class="sxs-lookup"><span data-stu-id="dc2ab-104">The **Scheduled work order maintenance jobs** page shows an overview of the work orders allocated to a resource.</span></span> <span data-ttu-id="dc2ab-105">Verkbeiðnir sem nota tilfangagerðirnar „Mannauður“, „Verkfæri“ og „Vél“ eru sýndar.</span><span class="sxs-lookup"><span data-stu-id="dc2ab-105">Work orders using resource types "Human resources", "Tool", and "Machine" are shown.</span></span> <span data-ttu-id="dc2ab-106">Til dæmis, ef viðhaldsstarfsmaður tilkynnir sig veikan er hægt að nota þessa síðu til að finna verkbeiðnir sem er úthlutað á starfskraftinn á fljótan hátt og úthluta síðan öðrum viðhaldsstarfsmanni til verksins.</span><span class="sxs-lookup"><span data-stu-id="dc2ab-106">For example, if a maintenance worker calls in sick, you can use this page to quickly find work orders allocated to the worker, and then allocate another maintenance worker to the job.</span></span>

1. <span data-ttu-id="dc2ab-108">Smelltu á **Eignastýringu** > **Sameiginlegt** > **Verkbeiðnir** > **Skipulagðar verkbeiðnir um viðhaldsverk**.</span><span class="sxs-lookup"><span data-stu-id="dc2ab-108">Click **Asset management** > **Common** > **Work orders** > **Scheduled work order maintenance jobs**.</span></span> <span data-ttu-id="dc2ab-109">Þú sérð lista yfir allar verkbeiðnir sem eru stilltar á líftímastöðu verkbeiðni „Skipulögð“ eða „Í vinnslu“.</span><span class="sxs-lookup"><span data-stu-id="dc2ab-109">You see a list of all work orders set to work order lifecycle state "Scheduled" or "In progress".</span></span>

2. <span data-ttu-id="dc2ab-110">Þú getur flokkað listann, til dæmis eftir viðhaldsstarfsmanni.</span><span class="sxs-lookup"><span data-stu-id="dc2ab-110">You can sort the list, for example, by maintenance worker.</span></span> <span data-ttu-id="dc2ab-111">Þú getur líka notað síuna til að takmarka listann til að birta verkbeiðnir sem eru úthlutaðar á tiltekin tilföng eða viðhaldsstarfsmann.</span><span class="sxs-lookup"><span data-stu-id="dc2ab-111">You can also use the filter to limit the list to display work orders allocated to a specific resource or maintenance worker.</span></span>

4. <span data-ttu-id="dc2ab-113">Ef þú vilt úthluta einum viðhaldsstarfsmanni á verkbeiðni skaltu velja verkbeiðnina og smella síðan á **Verkbeiðni**.</span><span class="sxs-lookup"><span data-stu-id="dc2ab-113">If you want to allocate one maintenance worker to a work order, select the work order, and then click **Work order**.</span></span>

5. <span data-ttu-id="dc2ab-114">Síðan **Verkbeiðni** opnast.</span><span class="sxs-lookup"><span data-stu-id="dc2ab-114">The **Work order** page opens.</span></span> <span data-ttu-id="dc2ab-115">Smelltu á **Senda** til að skipuleggja verkbeiðnina til ákveðins viðhaldsstarfsmanns.</span><span class="sxs-lookup"><span data-stu-id="dc2ab-115">Click **Dispatch** to schedule the work order to a specific maintenance worker.</span></span>
