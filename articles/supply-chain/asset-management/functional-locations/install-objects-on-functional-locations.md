# <a name="install-assets-on-functional-locations"></a><span data-ttu-id="41b80-103">Setja upp eignir á virkum staðsetningum</span><span class="sxs-lookup"><span data-stu-id="41b80-103">Install assets on functional locations</span></span>

[!include [banner](../../includes/banner.md)]

 

<span data-ttu-id="41b80-104">Eftir að þú hefur búið til skipulag virkra staðsetninga er næsta skref að setja upp eignir á viðkomandi virkum staðsetningum.</span><span class="sxs-lookup"><span data-stu-id="41b80-104">After you've created functional location structures, the next step is to install assets on the relevant functional locations.</span></span> <span data-ttu-id="41b80-105">Þetta efni útskýrir hvernig á að setja upp eignir á þessum virku staðsetningum í eignastjórnun.</span><span class="sxs-lookup"><span data-stu-id="41b80-105">This topic explains how to install assets on those functional locations in Asset Management.</span></span> <span data-ttu-id="41b80-106">Nánari upplýsingar um hvernig stofna eigi aðallykla, sjá [Kynning á eignum](../objects/introduction-to-objects.md).</span><span class="sxs-lookup"><span data-stu-id="41b80-106">For information about how to create assets, see [Introduction to assets](../objects/introduction-to-objects.md).</span></span>

<span data-ttu-id="41b80-107">Ef þú hefur búið til eignaskipulag verður að setja upp alla eignaskipan á virka staðsetningu.</span><span class="sxs-lookup"><span data-stu-id="41b80-107">If you've created an asset structure, the whole asset structure must be installed on a functional location.</span></span> <span data-ttu-id="41b80-108">Þess vegna er aðeins hægt að velja yfireignir (efstu eignir sem hafa enga yfireign) á virkri staðsetnigu.</span><span class="sxs-lookup"><span data-stu-id="41b80-108">Therefore, only parent assets (top-level assets that have no parent asset) can be selected on a functional location.</span></span> <span data-ttu-id="41b80-109">Allar tengdar undireignir (undireignir) verða einnig settar upp á virkri staðsetningu.</span><span class="sxs-lookup"><span data-stu-id="41b80-109">All related child assets (sub-assets) will also be installed on the functional location.</span></span> <span data-ttu-id="41b80-110">Þegar þú setur upp eignir á virkri staðsetningu, gætu fjárhagsvíddir virkrar staðsetningar verið fluttar sjálfkrafa yfir á þær, allt eftir uppsetningu á gerð virkrar staðsetningar sem er valin fyrir virka staðsetningu.</span><span class="sxs-lookup"><span data-stu-id="41b80-110">When you install assets on a functional location, the financial dimensions of the functional location might be automatically transferred to them, depending on the setup on the functional location type that is selected for the functional location.</span></span> <span data-ttu-id="41b80-111">Nánari upplýsingar um hvernig setja skal upp gerðir virkra staðsetinga er að finna í [Gerðir virkra staðsetninga](../setup-for-functional-locations/functional-location-types.md).</span><span class="sxs-lookup"><span data-stu-id="41b80-111">For more information about how to set up functional location types, see [Functional location types](../setup-for-functional-locations/functional-location-types.md).</span></span>

> <span data-ttu-id="41b80-112">Þú getur sett upp eignategundir á gerð virkrar staðsetningar sem er notuð fyrir virka staðsetningu.</span><span class="sxs-lookup"><span data-stu-id="41b80-112">You can set up asset types on the functional location type that is used for a functional location.</span></span> <span data-ttu-id="41b80-113">Í þessu tilfelli, þegar þú setur upp eignir á virkri staðsetningu eru aðeins yfireignir sem eru með sömu eignategund sýndar á listanum yfir eignir sem hægt er að setja upp á virkri staðsetningu.</span><span class="sxs-lookup"><span data-stu-id="41b80-113">In this case, when you install assets on the functional location, only parent assets that have the same asset type are shown in the list of assets that can be installed on the functional location.</span></span>

<span data-ttu-id="41b80-114">Eftir að þú hefur sett upp eignir á virkri staðsetningu geturðu skipt út yfireign eða eignaskipulagi eins og þú þarft.</span><span class="sxs-lookup"><span data-stu-id="41b80-114">After you've installed assets on a functional location, you can replace a parent asset or an asset structure as you require.</span></span> <span data-ttu-id="41b80-115">Eins og þegar þú setur upp eignir velurðu yfireign sem á að skipta um.</span><span class="sxs-lookup"><span data-stu-id="41b80-115">As when you install assets, you select a parent asset to replace.</span></span> <span data-ttu-id="41b80-116">Einnig verður skipt út öllum tengdum undireignum.</span><span class="sxs-lookup"><span data-stu-id="41b80-116">All related child assets will also be replaced.</span></span> 


## <a name="install-an-asset-structure-on-a-functional-location"></a><span data-ttu-id="41b80-117">Settu upp eignaskipulag á virkri staðsetningu</span><span class="sxs-lookup"><span data-stu-id="41b80-117">Install an asset structure on a functional location</span></span>

1. <span data-ttu-id="41b80-118">Veldu **Eignastýring** \> **Sameiginlegt** \> **Virkar staðsetningar** \> **Allar virkar staðsetningar** eða **Virkar virkar staðsetnigar**.</span><span class="sxs-lookup"><span data-stu-id="41b80-118">Select **Asset management** \> **Common** \> **Functional locations** \> **All Functional locations** or **Active functional locations**.</span></span>
2. <span data-ttu-id="41b80-119">Veldu virka staðsetningu til að setja upp eign á.</span><span class="sxs-lookup"><span data-stu-id="41b80-119">Select the functional location to install an asset on.</span></span>
3. <span data-ttu-id="41b80-120">Velja **Setja upp eign**.</span><span class="sxs-lookup"><span data-stu-id="41b80-120">Select **Install asset**.</span></span>

    <span data-ttu-id="41b80-121">Hlutinn **Eigindir** sýnir lista yfir eigindakröfur sem eru settar upp á gerð virkrar staðsetningar sem er valin fyrir virka staðsetningu.</span><span class="sxs-lookup"><span data-stu-id="41b80-121">The **Attributes** section shows a list of the asset attribute requirements that are set up on the functional location type that is selected for the functional location.</span></span> <span data-ttu-id="41b80-122">Egindirnar eru eingöngu ætlað til upplýsinga.</span><span class="sxs-lookup"><span data-stu-id="41b80-122">The attributes are for informational purposes only.</span></span> <span data-ttu-id="41b80-123">Kerfið staðfestir ekki eigindirnar gagnvart eignareigindunum sem eru settir upp á eigninni sem þú ert að setja upp.</span><span class="sxs-lookup"><span data-stu-id="41b80-123">The system doesn't validate the attributes against the asset attributes that are set up on the asset that you're installing.</span></span> <span data-ttu-id="41b80-124">Þú verður að gera þá staðfestingu eftir að þú hefur valið eign í reitnum **Eignir**.</span><span class="sxs-lookup"><span data-stu-id="41b80-124">You must do that validation after you select an asset in the **Asset** field.</span></span>

4. <span data-ttu-id="41b80-125">Í reitnum **Eignir** veldu yfireignina sem á að setja upp.</span><span class="sxs-lookup"><span data-stu-id="41b80-125">In the **Asset** field, select the parent asset to install.</span></span> <span data-ttu-id="41b80-126">Allar tengdar undireignir fylgja sjálfkrafa með uppsetningunni.</span><span class="sxs-lookup"><span data-stu-id="41b80-126">All related child assets are automatically included in the installation.</span></span>

    <span data-ttu-id="41b80-127">Hlutinn **Eigindir eigna** til hægri á eignalistanum sýnir eignareigindirnar sem tengjast völdum eigninni.</span><span class="sxs-lookup"><span data-stu-id="41b80-127">The **Asset attributes** section to the right of the asset list shows the asset attributes that are related to the selected asset.</span></span>

5. <span data-ttu-id="41b80-128">Í reitnum **Virkt**, veldu dagsetningu og tíma sem eignauppsetningin gildir frá.</span><span class="sxs-lookup"><span data-stu-id="41b80-128">In the **Effective** field, select the date and time that the asset installation is valid from.</span></span> <span data-ttu-id="41b80-129">Eftir þann dag og tíma mun kostnaður vegna eignarinnar og tengdar undireignir tengjast virkni staðsetningunni.</span><span class="sxs-lookup"><span data-stu-id="41b80-129">After that date and time, costs for the asset and related sub-assets will be related to the functional location.</span></span>

    > [!NOTE]
    > <span data-ttu-id="41b80-130">Eignaeigindunum sem eru settar upp á eigninni er bætt við hlutann **Eigindir**.</span><span class="sxs-lookup"><span data-stu-id="41b80-130">The asset attributes that are set up on the asset are added to the **Attributes** section.</span></span> <span data-ttu-id="41b80-131">Til dæmis, eigindakröfunni **Þyngd** hefur verið bætt við sem krafa bæði á eignina og virkri staðsetningu.</span><span class="sxs-lookup"><span data-stu-id="41b80-131">For example, the **Weight** attribute requirement has been added as a requirement on both the asset and the functional location.</span></span> <span data-ttu-id="41b80-132">Ef eignin hefur eigindakröfur af sömu gerð og virk staðsetningin, eru gildi eigindaskilyrða eignarinnar færð inn í reitnum **Gildi**.</span><span class="sxs-lookup"><span data-stu-id="41b80-132">If the asset has attribute requirements of the same type as the functional location, the values of the asset's attribute requirements are entered in the **Value** fields.</span></span> <span data-ttu-id="41b80-133">Þess vegna getur þú staðfest eignaverðin gagnvart eiginleikakröfunum sem settar eru upp á virkri staðsetningu.</span><span class="sxs-lookup"><span data-stu-id="41b80-133">Therefore, you can validate the asset values against the attribute requirements that are set up on the functional location.</span></span> <span data-ttu-id="41b80-134">Eigindakröfur sem settar eru upp á virkni staðsetningunni eru merktar með hakmerki.</span><span class="sxs-lookup"><span data-stu-id="41b80-134">The attribute requirements that are set up on the functional location are marked with a check mark.</span></span>

    > [!NOTE]
    > <span data-ttu-id="41b80-136">Fylgdu skrefum 1 til 6 í þessu ferli til að breyta uppsetningu á eign með því að setja hana upp á nýjan virkan stað.</span><span class="sxs-lookup"><span data-stu-id="41b80-136">To change the installation of an asset by installing it on a new functional location, follow steps 1 through 6 of this procedure.</span></span> <span data-ttu-id="41b80-137">Þegar þú setur upp eign á nýjan virkan stað er eignin sjálfkrafa fjarlægð frá fyrri virkni stað.</span><span class="sxs-lookup"><span data-stu-id="41b80-137">When you install an asset on a new functional location, the asset is automatically uninstalled from the previous functional location.</span></span> <span data-ttu-id="41b80-138">Allar virkar viðhaldsbeiðnir eða vinnupantanir sem voru búnar til á eigninni áður en þú settir hana upp á nýjan virkan stað er **ekki** sjálfkrafa flutt á nýja virkni stað.</span><span class="sxs-lookup"><span data-stu-id="41b80-138">Any active maintenance requests or work orders that were created on the asset before you installed it on a new functional location are **not** automatically transferred to the new functional location.</span></span> <span data-ttu-id="41b80-139">Ef þessar viðhaldsbeiðnir og verkbeiðnir eru enn nauðsynlegar fyrir eignina, verður þú að búa þær til handvirkt eftir að eignin er sett upp á nýjum stað.</span><span class="sxs-lookup"><span data-stu-id="41b80-139">If those maintenance requests and work orders are still required for the asset, you must manually re-create them after the asset is installed on the new location.</span></span>

7. <span data-ttu-id="41b80-140">Til að skoða lista yfir allar eignir, þar á meðal undireignir, sem eru settar upp á starfræna staðsetningu, velurðu virkni staðsetningu á síðunni **Allar virkar staðsetningar** og veldu síðan **Eignir**.</span><span class="sxs-lookup"><span data-stu-id="41b80-140">To view a list of all the assets, including sub-assets, that are installed on the functional location, select the functional location on the **All Functional locations** page, and then select **Assets**.</span></span>
8. <span data-ttu-id="41b80-141">Til að skoða lista yfir virkar viðhaldsbeiðnir, virkar verkbeiðnir eða bilanaskráningar sem tengjast eignunum sem eru settar upp á virkri staðsetningu skaltu velja virku staðsetninguna á síðunni **Allar virkar staðsetningar** og veldu síðan **Beiðnir**, **Verkbeiðnir**, eða **Bilanir**.</span><span class="sxs-lookup"><span data-stu-id="41b80-141">To view a list of active maintenance requests, active work orders, or fault registrations that are related to the assets that are installed on a functional location, select the functional location on the **All Functional locations** page, and then select **Requests**, **Work orders**, or **Faults**.</span></span>

> [!NOTE]
> <span data-ttu-id="41b80-142">Þegar eignatengdum gögnum er breytt eru þau uppfærð sjálfkrafa á virkri staðsetningu sem eignin er sett upp á.</span><span class="sxs-lookup"><span data-stu-id="41b80-142">When asset-related data is changed, it's automatically updated on the functional location that the asset is installed on.</span></span> <span data-ttu-id="41b80-143">Þessi sjálfvirka uppfærsla lýtur að breytingum á viðhaldsbeiðnum, verkbeiðnum, skráningum á eignabilunum, skráningum á niðurtíma vegna viðhalds og skráningum eignamælinga.</span><span class="sxs-lookup"><span data-stu-id="41b80-143">This automatic update pertains to changes to maintenance requests, work orders, asset fault registrations, maintenance downtime registrations, and asset measure registrations.</span></span>
