# <a name="functional-location-lifecycle-states"></a><span data-ttu-id="148e2-103">Líftímastöður virkra staðsetninga</span><span class="sxs-lookup"><span data-stu-id="148e2-103">Functional location lifecycle states</span></span>

[!include [banner](../../includes/banner.md)]

 

<span data-ttu-id="148e2-104">Þetta efnisatriði lýsir því hvernig á að setja upp líftímastöður og líftímalíkön virkra staðsetninga í eignastjórnun.</span><span class="sxs-lookup"><span data-stu-id="148e2-104">This topic describes how to set up functional location lifecycle states and lifecycle models in Asset Management.</span></span> <span data-ttu-id="148e2-105">Líftímastöður virkra staðsetninga skilgreina þau ríki sem hagnýtur staðsetning getur farið í gegnum, til dæmis búið til, virkan og endað.</span><span class="sxs-lookup"><span data-stu-id="148e2-105">Functional location lifecycle states define the states that a functional location can go through, for example, created, active, and ended.</span></span> <span data-ttu-id="148e2-106">Þú getur skoðað allar virkar staðsetningar, óháð líftímastöðu þeirra, á listasíðunni **Allar virkar staðsetningar**.</span><span class="sxs-lookup"><span data-stu-id="148e2-106">You are able to view all functional locations, regardless of their lifecycle state, in the **All functional locations** list page.</span></span> <span data-ttu-id="148e2-107">Þú getur breytt stöðu virkrar staðsetningar með því að velja hana á listasíðunni **Allar virkar staðsetningar** og velja **Uppfæra stöðu virkar staðsetningar**.</span><span class="sxs-lookup"><span data-stu-id="148e2-107">You can change the state of a functional location by selecting it in the **All functional locations** list page and selecting **Update functional location state**.</span></span>

## <a name="set-up-functional-location-lifecycle-states"></a><span data-ttu-id="148e2-108">Setja upp líftímastöður virkra staðsetninga</span><span class="sxs-lookup"><span data-stu-id="148e2-108">Set up functional location lifecycle states</span></span>

1. <span data-ttu-id="148e2-109">Veldu **Eignastýring** > **Uppsetning** > **Virkar staðsetningar** > **Líftímastöður**.</span><span class="sxs-lookup"><span data-stu-id="148e2-109">Select **Asset management** > **Setup** > **Functional locations** > **Lifecycle states**.</span></span>
2. <span data-ttu-id="148e2-110">Veldu **Nýr** til að stofna nýja virk staðsetning.</span><span class="sxs-lookup"><span data-stu-id="148e2-110">Select **New** to create a new functional location state.</span></span>
3. <span data-ttu-id="148e2-111">Settu stöðukennið inn í reitinn **Líftímastaða** og heiti á stöðu virkrar staðsetningar í reitnum **Heiti**.</span><span class="sxs-lookup"><span data-stu-id="148e2-111">Insert the state ID in the **Lifecycle state** field and a name for the functional location state in the **Name** field.</span></span> <span data-ttu-id="148e2-112">Í reitnum **Líftímalíkön** er hægt að sjá fjölda virkra líftímalíkana virkrar staðsetningar sem nota stöðu virkrar staðsetningar.</span><span class="sxs-lookup"><span data-stu-id="148e2-112">In the **Lifecycle models** field, you can see the number of functional location lifecycle models that uses the functional location state.</span></span>
4. <span data-ttu-id="148e2-113">Á flýtiflipanum **Almennt** velurðu „Já“ á skiptihnappnum **Virkur** ef virk staðsetning ætti að vera virk í þessari stöðu.</span><span class="sxs-lookup"><span data-stu-id="148e2-113">On the **General** FastTab, select "Yes" on the **Active** toggle button if the functional location should be active at this state.</span></span>
5. <span data-ttu-id="148e2-114">Veldu „Já“ á skiptihnappnum **Stofna eignir** ef það ætti að vera mögulegt að búa sjálfkrafa til eign með sama nafni og virka staðsetningin og setja hana upp á virkri staðsetningu í þessari stöðu.</span><span class="sxs-lookup"><span data-stu-id="148e2-114">Select "Yes" on the **Create assets** toggle button if it should be possible to automatically create an asset with the same name as the functional location and install it on the functional location at this state.</span></span>  
>[!NOTE]
><span data-ttu-id="148e2-115">Þessi skiptihnappur tengist reitnum **Gerð eigna** á flýtiflipanum **Almennt** í forminu **Gerðir virkra staðsetninga** (**Eignastýring** > **Uppsetning** > **Virkar staðsetningar** > **Gerðir virkra staðsetninga**).</span><span class="sxs-lookup"><span data-stu-id="148e2-115">This toggle button relates to the **Asset type** field on the **General** FastTab in the **Functional location types** form (**Asset management** > **Setup** > **Functional locations** > **Functional location types**).</span></span>
6. <span data-ttu-id="148e2-116">Veldu „Já“ á skiptihnappnum **Endurnefna staðsetningu** ef það ætti að vera mögulegt að breyta heiti á virkni staðsetningunni í þessari stöðu.</span><span class="sxs-lookup"><span data-stu-id="148e2-116">Select "Yes" on the **Rename location** toggle button if it should be possible to change the name of the functional location at this state.</span></span>
7. <span data-ttu-id="148e2-117">Veldu „Já“ á skiptihnappnum **Nýjar undirstaðsetningar** ef það ætti að vera mögulegt að bæta nýjum undirstaðsetningum við virku staðsetninguna í þessari stöðu.</span><span class="sxs-lookup"><span data-stu-id="148e2-117">Select "Yes" on the **New sub locations** toggle button if it should be possible to add new sub locations to the functional location at this state.</span></span>
8. <span data-ttu-id="148e2-118">Veldu „Já“ á skiptihnappnum **Setja upp eignir** ef það ætti að vera mögulegt að setja upp eignir á virku staðsetningunni í þessari stöðu.</span><span class="sxs-lookup"><span data-stu-id="148e2-118">Select "Yes" on the **Install assets** toggle button if it should be possible to install assets on the functional location at this state.</span></span>
9. <span data-ttu-id="148e2-119">Veldu „Já“ á skiptihnappnum **Eyða virkri staðsetningu** ef það ætti að vera mögulegt að eyða virkri staðsetningu í þessari stöðu.</span><span class="sxs-lookup"><span data-stu-id="148e2-119">Select "Yes" on the **Delete functional location** toggle button if it should be possible to delete the functional location at this state.</span></span>
10. <span data-ttu-id="148e2-120">Veldu eignarstöðu í reitnum **Líftímastaða** ef þú vilt að líftímastaða eignar fyrir allar eignir sem settar eru upp á virkri staðsetningu verði uppfærðar sjálfkrafa í þessari stöðu.</span><span class="sxs-lookup"><span data-stu-id="148e2-120">Select an asset state in the **Lifecycle state** field if you want the asset lifecycle state for all assets installed on the functional location to be automatically updated at this state.</span></span> <span data-ttu-id="148e2-121">Dæmi: Ef þú lokar virkri staðsetningu og stillir líftímastöðu virkrar staðsetningar á „Lokið“, gætirðu viljað breyta sjálfkrafa líftímastöðu eigna sem eru settar upp á þeirri virku staðsetningu í „Ekki í notkun“.</span><span class="sxs-lookup"><span data-stu-id="148e2-121">Example: If you close down a functional location, and set the functional location lifecycle state to "Ended", you may want to automatically change the lifecycle state of the assets installed on that functional location to "Not in use".</span></span>


>[!NOTE]
><span data-ttu-id="148e2-122">Líftímastaða virkrar staðsetningar, líftímalíkön og gerðir eru tengdar og notaðar á sama hátt og líftímastöður verkbeiðni, líftímalíkön verkbeiðni og gerðir verkbeiðni.</span><span class="sxs-lookup"><span data-stu-id="148e2-122">Functional location lifecycle states, lifecycle models, and types are related and used in the same way as work order lifecycle states, work order lifecycle models, and work order types.</span></span> 

## <a name="set-up-functional-location-lifecycle-models"></a><span data-ttu-id="148e2-123">Setja upp Líftímalíkön virkra staðsetninga</span><span class="sxs-lookup"><span data-stu-id="148e2-123">Set up functional location lifecycle models</span></span>

<span data-ttu-id="148e2-124">Þegar þú hefur búið til líftímastöður sem krafist er fyrir virkar staðsetningar þínar, þeim er hægt að skipta í hópa.</span><span class="sxs-lookup"><span data-stu-id="148e2-124">When you have created the lifecycle states required for your functional locations, they can be divided into groups.</span></span> <span data-ttu-id="148e2-125">Þetta er gert til að búa til flæði líftímalíkansins sem hægt er að nota fyrir mismunandi gerðir af virkum stöðum.</span><span class="sxs-lookup"><span data-stu-id="148e2-125">This is done to create the lifecycle model flow that may be used for different types of functional locations.</span></span> <span data-ttu-id="148e2-126">Að lágmarki ætti að búa til eitt staðlað líftímalíkan fyrir virka staðsetningu.</span><span class="sxs-lookup"><span data-stu-id="148e2-126">As a minimum, one standard functional location lifecycle model should be created.</span></span>

1. <span data-ttu-id="148e2-127">Veldu **Eignastýring** > **Uppsetning** > **Virkar staðsetningar** > **Líftímalíkön**.</span><span class="sxs-lookup"><span data-stu-id="148e2-127">Select **Asset management** > **Setup** > **Functional locations** > **Lifecycle models**.</span></span>
2. <span data-ttu-id="148e2-128">Veldu **Nýr** til að stofna nýtt líftímalíkan.</span><span class="sxs-lookup"><span data-stu-id="148e2-128">Select **New** to create a new lifecycle model.</span></span>
3. <span data-ttu-id="148e2-129">Settu líftímalíkanið inn í reitinn **Líftímalíkan** og heiti á líftímalíkaninu í reitnum **Heiti**.</span><span class="sxs-lookup"><span data-stu-id="148e2-129">Insert the lifecycle model ID in the **Lifecycle model** field and a name for the lifecycle model in the **Name** field.</span></span> <span data-ttu-id="148e2-130">Í reitunum **Gerðir virkra staðsetninga** og **Líftímastöður** geturðu séð fjölda virka staðsetninga sem nota líftímalíkanið og fjölda valinna staða í líftímalíkaninu.</span><span class="sxs-lookup"><span data-stu-id="148e2-130">In the **Functional location types** and **Lifecycle states** fields, you can see the number of functional location types that uses the lifecycle model and the number of states selected in the lifecycle model.</span></span>
4. <span data-ttu-id="148e2-131">Á flýtiflipanum **Líftímastöður** velurðu þær stöður sem ætti að vera með í líkaninu.</span><span class="sxs-lookup"><span data-stu-id="148e2-131">On the **Lifecycle states** FastTab, select the states that should be included in the model.</span></span> <span data-ttu-id="148e2-132">Þetta er gert með því að smella á stöðuna í hlutanum **Eftirstandandi líftímastöður** og smella á hnappinn ![áfram ör](media/02-setup-for-functional-locations.png).</span><span class="sxs-lookup"><span data-stu-id="148e2-132">This is done by clicking on a state in the **Lifecycle states remaining** section and clicking the ![forward arrow](media/02-setup-for-functional-locations.png) button.</span></span>
5. <span data-ttu-id="148e2-133">Ef þú vilt velja allar tiltækar stöður fyrir líkan, smelltu á hnappinn ![veldu öll tiltæk stig](media/03-setup-for-functional-locations.png).</span><span class="sxs-lookup"><span data-stu-id="148e2-133">If you want to select all the available states for a model, click the ![select all available stages](media/03-setup-for-functional-locations.png) button.</span></span> <span data-ttu-id="148e2-134">Allar stöður eru fluttar í hlutann **Valdar líftímastöður**.</span><span class="sxs-lookup"><span data-stu-id="148e2-134">All states are transferred to the **Lifecycle states selected** section.</span></span>
6. <span data-ttu-id="148e2-135">Ef þú vilt fjarlægja valda stöðu úr líkaninu skaltu velja stöðuna í hlutanum **Valdar líftímastöður** og velja síðan hnappinn ![aftur ör](media/04-setup-for-functional-locations.png).</span><span class="sxs-lookup"><span data-stu-id="148e2-135">If you want to remove a selected state from the model, select the state in the **Lifecycle states selected** section and then select the ![back arrow](media/04-setup-for-functional-locations.png) button.</span></span>
7. <span data-ttu-id="148e2-136">Veldu **Uppfærslur á líftímastöðu** til að skilgreina hvaða líftímastaða getur fylgt valinni stöðu.</span><span class="sxs-lookup"><span data-stu-id="148e2-136">Select **Lifecycle state updates** to define which lifecycle states can follow a selected state.</span></span>
