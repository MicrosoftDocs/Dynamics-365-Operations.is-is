# <a name="create-a-bar-code-for-a-product"></a><span data-ttu-id="6d8e5-103">Stofna strikamerki fyrir afurð</span><span class="sxs-lookup"><span data-stu-id="6d8e5-103">Create a bar code for a product</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="6d8e5-104">Þetta efni sýnir hvernig til dæmis strikamerki með vörunúmerið M0001 er stofnað handvirkt.</span><span class="sxs-lookup"><span data-stu-id="6d8e5-104">This topic shows how to manually create a bar code using the item number M0001 as an example.</span></span> <span data-ttu-id="6d8e5-105">Sýnigögn fyrirtækisins til að stofna þetta ferli er USMF.</span><span class="sxs-lookup"><span data-stu-id="6d8e5-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="6d8e5-106">Veldu **Viðhald vöru** á heimasíðunni.</span><span class="sxs-lookup"><span data-stu-id="6d8e5-106">Select **Released product maintenance** on the homepage.</span></span>
2. <span data-ttu-id="6d8e5-107">Farðu í **Vörur > Útgefnar vörur** undir hlutanum **Tenglar**.</span><span class="sxs-lookup"><span data-stu-id="6d8e5-107">Go to **Products > Released products** under the **Links** section.</span></span>
3. <span data-ttu-id="6d8e5-108">Í listanum skal finna og velja þá skráningu sem óskað er eftir.</span><span class="sxs-lookup"><span data-stu-id="6d8e5-108">In the list, find and select the desired record.</span></span> <span data-ttu-id="6d8e5-109">Í þessu dæmi skal velja vörunúmerið **M0001.**</span><span class="sxs-lookup"><span data-stu-id="6d8e5-109">For this example, select item number **M0001**.</span></span>
4. <span data-ttu-id="6d8e5-110">Á Aðgerðasvæðinu skal velja **Stjórna birgðum.**</span><span class="sxs-lookup"><span data-stu-id="6d8e5-110">On the Action Pane, select **Manage inventory**.</span></span>
5. <span data-ttu-id="6d8e5-111">Veldu **Strikamerki**.</span><span class="sxs-lookup"><span data-stu-id="6d8e5-111">Select **Bar codes**.</span></span>
6. <span data-ttu-id="6d8e5-112">Veljið **Nýtt**.</span><span class="sxs-lookup"><span data-stu-id="6d8e5-112">Select **New**.</span></span>
7. <span data-ttu-id="6d8e5-113">Merktu valda röð sem er búin til á listanum hér að neðan.</span><span class="sxs-lookup"><span data-stu-id="6d8e5-113">Mark the selected row that is created in the list below.</span></span>
8. <span data-ttu-id="6d8e5-114">Í reitnum **Uppsetning strikamerkis** slærðu inn eða velur gildi.</span><span class="sxs-lookup"><span data-stu-id="6d8e5-114">In the **Barcode setup** field, enter or select a value.</span></span>
9. <span data-ttu-id="6d8e5-115">Í reitnum **Strikamerki** slærðu inn eða velur gildi.</span><span class="sxs-lookup"><span data-stu-id="6d8e5-115">In the **Bar code** field, enter or select a value.</span></span>
10. <span data-ttu-id="6d8e5-116">Í reitnum **Strikamerki** slærðu inn gildi.</span><span class="sxs-lookup"><span data-stu-id="6d8e5-116">In the **Bar code** field, type a value.</span></span>  
11. <span data-ttu-id="6d8e5-117">Í reitnum **Magn** slærðu inn tölu.</span><span class="sxs-lookup"><span data-stu-id="6d8e5-117">In the **Quantity** field, enter a number.</span></span>
12. <span data-ttu-id="6d8e5-118">Veljið **Vista**.</span><span class="sxs-lookup"><span data-stu-id="6d8e5-118">Select **Save**.</span></span>
13. <span data-ttu-id="6d8e5-119">Lokið síðunni.</span><span class="sxs-lookup"><span data-stu-id="6d8e5-119">Close the page.</span></span>
