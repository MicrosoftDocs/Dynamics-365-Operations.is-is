## <a name="credit-subscription-transactions"></a><span data-ttu-id="321a2-104">Kreditfæra áskriftarfærslur</span><span class="sxs-lookup"><span data-stu-id="321a2-104">Credit subscription transactions</span></span>

1.  <span data-ttu-id="321a2-105">Smelltu á **Þjónustustjórnun** \> **Almennt** \> **Þjónustuáskriftir** \> **Allar þjónustuáskriftir**.</span><span class="sxs-lookup"><span data-stu-id="321a2-105">Click **Service management** \> **Common** \> **Service subscriptions** \> **All service subscriptions**.</span></span>

2.  <span data-ttu-id="321a2-106">Veljið áskriftina sem tengd er áskriftarfærslunni sem stofna á kreditnótu fyrir.</span><span class="sxs-lookup"><span data-stu-id="321a2-106">Select the subscription attached to the subscription transaction for which you want to create a credit note.</span></span>

3.  <span data-ttu-id="321a2-107">Veldu **Sundurliða** flipann og smelltu síðan á **Þóknunarfærsla** hnappinn í aðgerðarúðunni.</span><span class="sxs-lookup"><span data-stu-id="321a2-107">Select the **Analyze** tab, and then click the **Fee transactions** button on the Action Pane.</span></span>

4.  <span data-ttu-id="321a2-108">Af **Þóknunarfærsla** skjámynd skaltu velja færsluna sem þú vilt búa til kreditnótu fyrir.</span><span class="sxs-lookup"><span data-stu-id="321a2-108">From the **Fee transactions** form, select the transaction for which you want to create a credit note.</span></span>

5.  <span data-ttu-id="321a2-109">Smelltu á **Aðgerðir** \> **Velja fyrir kreditnótu**.</span><span class="sxs-lookup"><span data-stu-id="321a2-109">Click **Functions** \> **Select for credit note**.</span></span>

6.  <span data-ttu-id="321a2-110">Frá **Velja fyrir kreditnótu** skjámynd skaltu velja færsluna sem þú vilt gjaldfæra og smella síðan á **Í lagi**.</span><span class="sxs-lookup"><span data-stu-id="321a2-110">From the **Select for credit note** form, select the transaction that you want to credit and then click **OK**.</span></span>


> [!NOTE]
> <P><span data-ttu-id="321a2-111">Þegar þú býrð til kreditnótuna skaltu ganga úr skugga um að þú veljir <STRONG>Kreditnótur</STRONG>.</span><span class="sxs-lookup"><span data-stu-id="321a2-111">When you create the credit note, make sure that you select <STRONG>Credit notes</STRONG>.</span></span> <span data-ttu-id="321a2-112">Þetta er að finna í <STRONG>Reikningsaðferð</STRONG> listanum í <STRONG>Búa til reikning</STRONG> svarglugganum.</span><span class="sxs-lookup"><span data-stu-id="321a2-112">This is found in the <STRONG>Invoicing method</STRONG> list in the <STRONG>Create invoice</STRONG> dialog box.</span></span></P>

<span data-ttu-id="321a2-113">Ef **Bakfærð uppsöfnun á gjaldfærslu** reiturinn í **Færibreytur Þjónustustjórnunar** skjámyndinni er stillt á **Handvirkt**, verður þú að bakfæra hverja uppsafnaða tekjufærslu fyrir sig áður en þú býrð til kreditnótutillögu fyrir færsluna.</span><span class="sxs-lookup"><span data-stu-id="321a2-113">If the **Reverse accruals on crediting** field in the **Service management parameters** form is set to **Manual**, you have to reverse each accrued revenue transaction individually before you create a credit note proposal for the transaction.</span></span>
