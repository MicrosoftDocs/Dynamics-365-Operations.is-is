<span data-ttu-id="8fb7c-104">[!taka með [borði](../includes/banner.md)</span><span class="sxs-lookup"><span data-stu-id="8fb7c-104">[!include [banner](../includes/banner.md)</span></span>


<span data-ttu-id="8fb7c-105">Þú getur búið til þjónustuáskrift frá verkefni í **Verkefnastjórnun og bókhald**.</span><span class="sxs-lookup"><span data-stu-id="8fb7c-105">You can create a service subscription from a project in **Project management and accounting**.</span></span>

1.  <span data-ttu-id="8fb7c-106">Smelltu á **Verkefnastjórnun og bókhald** \> **Almenn** \> **Verkefni** \> **Öll verkefni**.</span><span class="sxs-lookup"><span data-stu-id="8fb7c-106">Click **Project management and accounting** \> **Common** \> **Projects** \> **All projects**.</span></span> <span data-ttu-id="8fb7c-107">Opnaðu verkefnið sem þú vilt búa til áskrift fyrir.</span><span class="sxs-lookup"><span data-stu-id="8fb7c-107">Open the project that you want to create a subscription for.</span></span>

2.  <span data-ttu-id="8fb7c-108">Á **Verkefni** skjámynd, á **Aðgerðarúða**, á **Stjórna** flipanum, smelltu á **Þjónusta** \> **Þjónustuáskrift**.</span><span class="sxs-lookup"><span data-stu-id="8fb7c-108">On the **Projects** form, on the **Action Pane**, on the **Manage** tab, click **Service** \> **Service subscription**.</span></span>

3.  <span data-ttu-id="8fb7c-109">Sláðu inn einkvæmt auðkennisnúmer í **Áskrift** skjámynd í **Þjónustuáskrift** reitnum.</span><span class="sxs-lookup"><span data-stu-id="8fb7c-109">On the **Subscription** form, in the **Service subscription** field, enter a unique identification number.</span></span> <span data-ttu-id="8fb7c-110">Verknúmerið er sjálfkrafa fært inn í áskriftina fyrir valið verk.</span><span class="sxs-lookup"><span data-stu-id="8fb7c-110">The project number is entered automatically into the subscription for the selected project.</span></span>

4.  <span data-ttu-id="8fb7c-111">Færið inn upplýsingum í skjámyndinni.</span><span class="sxs-lookup"><span data-stu-id="8fb7c-111">Enter the remaining information in the form.</span></span>

<span data-ttu-id="8fb7c-112">Núna er hægt að stofna áskriftarfærslu fyrir áskriftina.</span><span class="sxs-lookup"><span data-stu-id="8fb7c-112">You can now create subscription transactions for the subscription.</span></span>

> <P><span data-ttu-id="8fb7c-113">Hafir þú sett upp númeraröð fyrir áskriftarflokkur er áskriftarkenni sjálfkrafa myndað fyrir áskriftinni þegar þú tiltekur áskriftarflokk.</span><span class="sxs-lookup"><span data-stu-id="8fb7c-113">If you have set up a number sequence for the subscription group, a subscription ID is generated automatically for the subscription when you specify the subscription group.</span></span></P>
