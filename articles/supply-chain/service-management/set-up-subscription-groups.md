# <a name="set-up-subscription-groups"></a><span data-ttu-id="f2248-103">Setja upp áskriftarflokka</span><span class="sxs-lookup"><span data-stu-id="f2248-103">Set up subscription groups</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="f2248-104">Smelltu á **Þjónustustjórnun** \> **Uppsetning** \> **Þjónusta áskrift** \> **Áskriftarflokkar** .</span><span class="sxs-lookup"><span data-stu-id="f2248-104">Click **Service management** \> **Setup** \> **Service subscriptions** \> **Subscription groups**.</span></span>

2.  <span data-ttu-id="f2248-105">Ýttu á Ctrl+N eða smelltu á **Nýtt** til að stofna nýjan áskriftarflokk.</span><span class="sxs-lookup"><span data-stu-id="f2248-105">Press Ctrl+N or click **New** to create a new subscription group.</span></span>

3.  <span data-ttu-id="f2248-106">Í reitinn **Áskriftarflokkur** skal slá inn heiti fyrir nýjan áskriftarflokk.</span><span class="sxs-lookup"><span data-stu-id="f2248-106">In the **Subscription group** field, enter a name for the new subscription group.</span></span>

5.  <span data-ttu-id="f2248-109">Til að safna tekjum vegna áskriftarfærslna skal velja gátreitinn **Safna upp tekjum** og velja síðan tíðni í reitnum **Uppsöfnun**.</span><span class="sxs-lookup"><span data-stu-id="f2248-109">To accrue revenue for subscription transactions, select the **Accrue revenue** check box, and then in the **Accrual** field, select a frequency.</span></span> <span data-ttu-id="f2248-110">Uppsöfnunartíðnin er byggð á tímabilskóða sem er settur upp í skjámyndinni **Tímabil**.</span><span class="sxs-lookup"><span data-stu-id="f2248-110">The accrual frequency is based on a period code that is set up in the **Periods** form.</span></span>
