# <a name="set-up-disposition-codes"></a><span data-ttu-id="cbe2a-103">Setja upp ráðstöfunarkóða</span><span class="sxs-lookup"><span data-stu-id="cbe2a-103">Set up disposition codes</span></span> 

[!include [banner](../includes/banner.md)]


<span data-ttu-id="cbe2a-104">Þú getur sett upp ráðstöfunarkóða til að tilgreina hvernig á að vinna úr vöru sem er skilað af viðskiptavini.</span><span class="sxs-lookup"><span data-stu-id="cbe2a-104">You can set up disposition codes to specify how to process an item that is returned by a customer.</span></span> <span data-ttu-id="cbe2a-105">Til dæmis, stofnað ráðstöfunarkóða með heitinu **Viðgerð og vöruskil** til að tilgreina að gert verði við vöruna sem skilað var og henni síðan skilað til viðskiptamannsins.</span><span class="sxs-lookup"><span data-stu-id="cbe2a-105">For example, create a disposition code named **Repair and return** to indicate that the returned item will be repaired and then returned to the customer.</span></span> <span data-ttu-id="cbe2a-106">Fyrir fleiri dæmi um ráðstöfunarkóða sem eru venjulega notaðir fyrir vörur sem er skilað af viðskiptavinum, sjá [Tilgreina hvað gera eigi við skilavörur](specify-how-to-dispose-of-returned-items.md).</span><span class="sxs-lookup"><span data-stu-id="cbe2a-106">For more examples of disposition codes that are typically used for items that are returned by customers, see [Specify how to dispose of returned items](specify-how-to-dispose-of-returned-items.md).</span></span>

<span data-ttu-id="cbe2a-107">Einnig er hægt að setja upp ástæðukóta til að aðstoða við að útskýra hvers vegna vöru var skilað.</span><span class="sxs-lookup"><span data-stu-id="cbe2a-107">You can also set up a reason code to help explain why an item was returned.</span></span> <span data-ttu-id="cbe2a-108">Nánari upplýsingar um ástæðukóða, sjá [Setja upp ástæðakóða](set-up-return-reason-code.md).</span><span class="sxs-lookup"><span data-stu-id="cbe2a-108">For more information about reason codes, see [Set up return reason codes](set-up-return-reason-code.md).</span></span>

1.  <span data-ttu-id="cbe2a-109">Veljið **Sala og markaðssetning** \> **Uppsetning** \> **Sölupantanir** \> **Skil** \> **Ráðstöfunarkóðar**.</span><span class="sxs-lookup"><span data-stu-id="cbe2a-109">Go to **Sales and marketing** \> **Setup** \> **Sales orders** \> **Returns** \> **Disposition codes**.</span></span>

2.  <span data-ttu-id="cbe2a-110">Veljið **Nýtt** til að stofna nýjan ráðstöfunarkóða.</span><span class="sxs-lookup"><span data-stu-id="cbe2a-110">Select **New** to create a new disposition code.</span></span>

3.  <span data-ttu-id="cbe2a-111">Færið inn einkvæmt lýsandi nafn, veljið aðgerð og færið inn lýsingu fyrir ráðstöfunarkóða.</span><span class="sxs-lookup"><span data-stu-id="cbe2a-111">Enter a unique, descriptive name, select an action, and enter a description for the disposition code.</span></span>

4.  <span data-ttu-id="cbe2a-112">Ef þú vilt tengja gjöld viðskiptavina við þennan ráðstöfunarkóða skal velja hnappinn **Gjöld** til að opna skjámyndina **Setja upp gjöld**.</span><span class="sxs-lookup"><span data-stu-id="cbe2a-112">If you want to associate any customer charges with this disposition code, select the **Charges** button to open the **Set up charges** form.</span></span>

5.  <span data-ttu-id="cbe2a-113">Ef þú vilt skilgreina einhverja ytri kóða sem til að passa við eigin ráðstöfunarkóða fyrirtækisins skal velja hnappinn **Ytri kóðar** til að opna skjámyndina **Ytri kóðar**.</span><span class="sxs-lookup"><span data-stu-id="cbe2a-113">If you want to define any external codes to match with the company's own disposition codes, select the **External codes** button to open the **External codes** form.</span></span>
