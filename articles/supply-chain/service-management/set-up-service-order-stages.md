# <a name="set-up-service-order-stages"></a><span data-ttu-id="0c83f-103">Setja upp stig þjónustupantana</span><span class="sxs-lookup"><span data-stu-id="0c83f-103">Set up service order stages</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="0c83f-104">Smelltu á **Þjónustustjórnun** \> **Uppsetning** \> **Þjónustupantanir** \> **Þjónustustig**.</span><span class="sxs-lookup"><span data-stu-id="0c83f-104">Click **Service management** \> **Setup** \> **Service orders** \> **Service stages**.</span></span>

2.  <span data-ttu-id="0c83f-105">Ýtið á CTRL+N til að stofna nýja færslu.</span><span class="sxs-lookup"><span data-stu-id="0c83f-105">Press CTRL+N to create a new record.</span></span>

3.  <span data-ttu-id="0c83f-106">Í reitunum **Þjónustustig** og **Lýsing** skal tilgreina auðkenni þjónustustigs og lýsingar.</span><span class="sxs-lookup"><span data-stu-id="0c83f-106">In the **Service stage** and **Description** fields, specify a service stage ID and description.</span></span>

4.  <span data-ttu-id="0c83f-107">Veldu viðeigandi færibreytur fyrir sviðið.</span><span class="sxs-lookup"><span data-stu-id="0c83f-107">Select the appropriate parameters for the stage.</span></span>

5.  <span data-ttu-id="0c83f-108">Veldu yfirstigið fyrir núverandi stig eða skildu reitinn **Yfireining** eftir auðan ef stigið er upphafsstigið í uppsetningu stigsins.</span><span class="sxs-lookup"><span data-stu-id="0c83f-108">Select the parent stage for the current stage or leave the **Parent** field blank if the stage is the initial stage in the stage setup.</span></span>


> [!NOTE]
> <P><span data-ttu-id="0c83f-109">Þú getur ekki breytt reitnum <STRONG>Yfireining</STRONG> eftir að þú hefur vistað stigið.</span><span class="sxs-lookup"><span data-stu-id="0c83f-109">You cannot modify the <STRONG>Parent</STRONG> field after you save the stage.</span></span> <span data-ttu-id="0c83f-110">Í staðinn er hægt að eyða færslunni og stofna hana aftur með öðru vali í reitnum <STRONG>Yfireining</STRONG>.</span><span class="sxs-lookup"><span data-stu-id="0c83f-110">Instead, you can delete the record and create the record again with a different selection in the <STRONG>Parent</STRONG> field.</span></span></P>
