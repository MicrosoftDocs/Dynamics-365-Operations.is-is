# <a name="set-up-service-intervals"></a><span data-ttu-id="fdca9-103">Setja upp þjónustubil</span><span class="sxs-lookup"><span data-stu-id="fdca9-103">Set up service intervals</span></span>  

[!include [banner](../includes/banner.md)]

<span data-ttu-id="fdca9-104">Þjónustubil gefur til kynna þá tíðni sem þjónustupöntunarlínur eru stofnaðar fyrir þjónustusamningslínur þegar þjónustupantanir eru stofnaðar.</span><span class="sxs-lookup"><span data-stu-id="fdca9-104">Service interval indicates the frequency with which service order lines are created for service agreement lines when you create service orders.</span></span>

1. <span data-ttu-id="fdca9-105">Smellið á **Þjónustustjórnun** \> **Uppsetning** \> **Þjónustusamningar** \> **Þjónustubil**.</span><span class="sxs-lookup"><span data-stu-id="fdca9-105">Click **Service management** \> **Setup** \> **Service agreements** \> **Service intervals**.</span></span>

3. <span data-ttu-id="fdca9-107">Færið inn kenni og lýsingu á þjónustubilinu.</span><span class="sxs-lookup"><span data-stu-id="fdca9-107">Enter the ID and description of the service interval.</span></span>
4. <span data-ttu-id="fdca9-108">Í reitnum **Svið** skal velja sviðið.</span><span class="sxs-lookup"><span data-stu-id="fdca9-108">In the **Range** field, select the range.</span></span>
5. <span data-ttu-id="fdca9-109">Í reitnum **Tíðni** skal slá inn tíðnina.</span><span class="sxs-lookup"><span data-stu-id="fdca9-109">In the **Frequency** field, type the frequency.</span></span> <span data-ttu-id="fdca9-110">Tíðnin er stuðullinn sem margfalda verður sviðið með til að fá út bil þjónustusamnings.</span><span class="sxs-lookup"><span data-stu-id="fdca9-110">The frequency is the factor by which you must multiply the range to obtain the interval for a service agreement.</span></span>
