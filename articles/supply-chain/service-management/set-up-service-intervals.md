# <a name="set-up-service-intervals"></a><span data-ttu-id="4b080-103">Setja upp þjónustubil</span><span class="sxs-lookup"><span data-stu-id="4b080-103">Set up service intervals</span></span>  

[!include [banner](../includes/banner.md)]

<span data-ttu-id="4b080-104">Þjónustubil gefur til kynna þá tíðni sem þjónustupöntunarlínur eru stofnaðar fyrir þjónustusamningslínur þegar þjónustupantanir eru stofnaðar.</span><span class="sxs-lookup"><span data-stu-id="4b080-104">Service interval indicates the frequency with which service order lines are created for service agreement lines when you create service orders.</span></span>

1. <span data-ttu-id="4b080-105">Smellið á **Þjónustustjórnun** \> **Uppsetning** \> **Þjónustusamningar** \> **Þjónustubil**.</span><span class="sxs-lookup"><span data-stu-id="4b080-105">Click **Service management** \> **Setup** \> **Service agreements** \> **Service intervals**.</span></span>
2. <span data-ttu-id="4b080-106">Stofnið nýtt þjónustubil.</span><span class="sxs-lookup"><span data-stu-id="4b080-106">Create a new service interval.</span></span>
3. <span data-ttu-id="4b080-107">Færið inn kenni og lýsingu á þjónustubilinu.</span><span class="sxs-lookup"><span data-stu-id="4b080-107">Enter the ID and description of the service interval.</span></span>
4. <span data-ttu-id="4b080-108">Í reitnum **Svið** skal velja sviðið.</span><span class="sxs-lookup"><span data-stu-id="4b080-108">In the **Range** field, select the range.</span></span>
5. <span data-ttu-id="4b080-109">Í reitnum **Tíðni** skal slá inn tíðnina.</span><span class="sxs-lookup"><span data-stu-id="4b080-109">In the **Frequency** field, type the frequency.</span></span> <span data-ttu-id="4b080-110">Tíðnin er stuðullinn sem margfalda verður sviðið með til að fá út bil þjónustusamnings.</span><span class="sxs-lookup"><span data-stu-id="4b080-110">The frequency is the factor by which you must multiply the range to obtain the interval for a service agreement.</span></span>
6. <span data-ttu-id="4b080-111">Ýtið á **Alt+S** til að vista þjónustubilið.</span><span class="sxs-lookup"><span data-stu-id="4b080-111">Press **Alt+S** to save the service interval.</span></span>
