<span data-ttu-id="b9292-105">Þegar birgðalokun er keyrð eru allar móttökur jafnaðar gagnvart sýndarúthreyfingum sem geyma heildar mótekið magn og virði.</span><span class="sxs-lookup"><span data-stu-id="b9292-105">When you run an inventory closing, all receipts are settled against a virtual issue, which holds the total received quantity and value.</span></span> <span data-ttu-id="b9292-106">Þessi sýndarúthreyfing hefur samsvarandi sýndarinnhreyfingu þaðan sem úthreyfingar eru jafnaðar.</span><span class="sxs-lookup"><span data-stu-id="b9292-106">This virtual issue has a corresponding virtual receipt from which the issues are settled.</span></span> <span data-ttu-id="b9292-107">Þannig fá allar úthreyfingar sama meðalkostnað.</span><span class="sxs-lookup"><span data-stu-id="b9292-107">In this manner, all issues get the same average cost.</span></span> <span data-ttu-id="b9292-108">Sýndarúthreyfinguna og -innhreyfinguna er hægt að sjá sem sýndarfærslu sem kallast flutningur fyrir vegið meðaltal birgðalokunar</span><span class="sxs-lookup"><span data-stu-id="b9292-108">The virtual issue and receipt can be seen as a virtual transfer, which is named the weighted average inventory closing transfer.</span></span>
