# <a name="create-a-location-profile"></a><span data-ttu-id="4895b-103">Stofna staðsetningarforstillingu</span><span class="sxs-lookup"><span data-stu-id="4895b-103">Create a location profile</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="4895b-104">Hver staðsetning í vöruhúsi verður að hafa staðsetningarforstillingu tengda sem lýsir eiginleikum staðsetningar, t.d. hvort staðsetning heimilar blandaðar vörur.</span><span class="sxs-lookup"><span data-stu-id="4895b-104">Every location in the warehouse needs to have a location profile associated with it that describes the properties of the location, for example, whether the location allows mixed items.</span></span> <span data-ttu-id="4895b-105">Í þessu ferli munum við stofna forstillingu fyrir staðsetningu sem ekki þarf stýringu númeraplötu.</span><span class="sxs-lookup"><span data-stu-id="4895b-105">In this procedure we’ll create a profile for a location that doesn’t require license plate control.</span></span> <span data-ttu-id="4895b-106">Við munum virkja blandaðar vörur og blandaða birgðastöðu, og leyfa reglulega talningu.</span><span class="sxs-lookup"><span data-stu-id="4895b-106">We’ll enable mixed items, and mixed inventory statuses, and allow cycle counting.</span></span> <span data-ttu-id="4895b-107">Hægt er að nota þetta ferli í sýnigögn fyrirtækisins USMF.</span><span class="sxs-lookup"><span data-stu-id="4895b-107">You can use this procedure in the USMF demo data company.</span></span>

1. <span data-ttu-id="4895b-108">Smellið á „Nýtt“.</span><span class="sxs-lookup"><span data-stu-id="4895b-108">Click New.</span></span>
2. <span data-ttu-id="4895b-109">Færa inn gildi í reitnum Kenni forstillingar staðsetningar.</span><span class="sxs-lookup"><span data-stu-id="4895b-109">In the Location profile ID field, type a value.</span></span>
3. <span data-ttu-id="4895b-110">Í reitinn Heiti skal slá inn gildi.</span><span class="sxs-lookup"><span data-stu-id="4895b-110">In the Name field, type a value.</span></span>
4. <span data-ttu-id="4895b-111">Sláðu inn eða veldu gildi í reitnum Staðsetningarsnið</span><span class="sxs-lookup"><span data-stu-id="4895b-111">In the Location format field, enter or select a value.</span></span>
5. <span data-ttu-id="4895b-112">Færa inn eða veljið gildi í svæðinu gerð staðsetningu.</span><span class="sxs-lookup"><span data-stu-id="4895b-112">In the Location type field, enter or select a value.</span></span>
6. <span data-ttu-id="4895b-113">Í reitinn Auðkenni fyrir forstillingu dreifingarstjórnunar skal slá inn eða velja gildi.</span><span class="sxs-lookup"><span data-stu-id="4895b-113">In the Dock management profile ID field, enter or select a value.</span></span>
7. <span data-ttu-id="4895b-114">Velja skal Já í svæðinu Leyfa blandaðar vörur.</span><span class="sxs-lookup"><span data-stu-id="4895b-114">Select Yes in the Allow mixed items field.</span></span>
8. <span data-ttu-id="4895b-115">Velja skal Já í svæðinu Leyfa blandað birgðastaða.</span><span class="sxs-lookup"><span data-stu-id="4895b-115">Select Yes in the Allow mixed  inventory statuses field.</span></span>
9. <span data-ttu-id="4895b-116">Velja skal Já í svæðinu Leyfa regluleg talning.</span><span class="sxs-lookup"><span data-stu-id="4895b-116">Select Yes in the Allow cycle counting field.</span></span>
10. <span data-ttu-id="4895b-117">Smellið á „Vista“.</span><span class="sxs-lookup"><span data-stu-id="4895b-117">Click Save.</span></span>
11. <span data-ttu-id="4895b-118">Fara í vöruhúsakerfi > Uppsetning > Vöruhús > Forstillingar staðsetningar.</span><span class="sxs-lookup"><span data-stu-id="4895b-118">Go to Warehouse management > Setup > Warehouse > Location profiles.</span></span>
