<span data-ttu-id="eb7e7-104">Vegna þess að Android er opinn uppruni verkefni, getur hvaða framleiðandi vélbúnaðar fyrir strikamerkjaskönnun vöruhúsa sem er búið til tæki til að keyra Android stýrikerfið.</span><span class="sxs-lookup"><span data-stu-id="eb7e7-104">Because Android is an open source project, any manufacturer of hardware for warehouse bar code scanners can build a device to run the Android operating system.</span></span> <span data-ttu-id="eb7e7-105">Tæki er aðeins Android-samhæft ef það getur keyrt forrit sem eru skrifuð fyrir Android framkvæmdarumhverfi.</span><span class="sxs-lookup"><span data-stu-id="eb7e7-105">A device is only Android-compatible if it can run apps that are written for the Android execution environment.</span></span>
