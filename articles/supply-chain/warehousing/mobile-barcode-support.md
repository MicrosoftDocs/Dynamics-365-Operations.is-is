<span data-ttu-id="0ef35-104">Vegna þess að Android er opinn uppruni verkefni, getur hvaða framleiðandi vélbúnaðar fyrir strikamerkjaskönnun vöruhúsa sem er búið til tæki til að keyra Android stýrikerfið.</span><span class="sxs-lookup"><span data-stu-id="0ef35-104">Because Android is an open source project, any manufacturer of hardware for warehouse bar code scanners can build a device to run the Android operating system.</span></span> <span data-ttu-id="0ef35-105">Tæki er aðeins Android-samhæft ef það getur keyrt forrit sem eru skrifuð fyrir Android framkvæmdarumhverfi.</span><span class="sxs-lookup"><span data-stu-id="0ef35-105">A device is only Android-compatible if it can run apps that are written for the Android execution environment.</span></span>
<span data-ttu-id="0ef35-106">Hins vegar getur vélbúnaðarsöluaðili breytt og búið til yfirborð fyrir Android-útgáfuna sem er keyrð á vélbúnaði hans.</span><span class="sxs-lookup"><span data-stu-id="0ef35-106">However, a hardware vendor can modify and create overlays for the Android version that runs on their hardware.</span></span> <span data-ttu-id="0ef35-107">Microsoft getur ekki tekið neina ábyrgð á því að tryggja að strikamerki skannaforrit fyrir Android sé samhæft við strikamerki skanna vélbúnaðar framleiðanda og Android-útgáfuna sem er keyrð á henni.</span><span class="sxs-lookup"><span data-stu-id="0ef35-107">Microsoft cannot take any responsibility to ensure that a mobile bar code scanning app for Android is compatible with a manufacturer’s bar code scanning hardware and the Android version that runs on it.</span></span>

<span data-ttu-id="0ef35-110">Sem viðskiptavinur, mælum við með því að þú prófir skönnunarforrit farsíma fyrir vöruhús á völdum vélbúnaði áður en þú ákveður hvaða vélbúnað þú vilt kaupa.</span><span class="sxs-lookup"><span data-stu-id="0ef35-110">As a customer, we recommend that you test the Warehouse mobile scanning app on selected hardware before you decide on the hardware that you want to buy.</span></span>
