# <a name="resource-name-isnt-updated-after-changing-a-worker-name-in-the-address-book"></a>Tilfangaheiti er ekki uppfært eftir að nafni starfskrafts er breytt í aðsetursbók

## <a name="symptoms"></a>Einkenni

Ef nafni starfskrafts er breytt í altæku aðsetursbókinni er samsvarandi heiti ekki uppfært í aðaltilfangaflokki.

## <a name="resolution"></a>Lausn

Samsvarandi tilfangaheiti er ekki uppfært sjálfkrafa eins og er. Til að laga vandamálið þarf að uppfæra heiti tilfanga handvirkt.
