# <a name="cant-end-a-production-order-because-of-a-bom-consumptioncost-value-error"></a>Get ekki lokið framleiðslupöntun vegna villu í gildi fyrir consumptionCost uppskriftar

## <a name="symptoms"></a>Einkenni

Þegar reynt er að ljúka framleiðslupöntun færðu eftirfarandi villuboð:

> Gildið við útreikning consumptionCost uppskriftar verður að vera neikvætt við úthreyfingu birgða.

## <a name="resolution"></a>Lausn

Þetta vandamál var lagfært í útgáfu 10.0.15.
