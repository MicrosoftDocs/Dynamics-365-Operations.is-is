# <a name="invalid-license-plate-or-location-error-when-scanning-in-the-mobile-app"></a>Ógild númeraplata eða staðsetningarvilla við skönnun í farsímaforriti

## <a name="symptoms"></a>Einkenni

Þegar þú skannar númeraplötukenni eða staðsetningu gætir þú fengið upp eftirfarandi villuboð:

> Númeraplatan eða staðsetningin er ekki gild.

## <a name="resolution"></a>Lausn

Gangið úr skugga um að kenni númeraplötunnar sé ekki tekið frá eitthvað annað. Þetta vandamál kom upp þegar gildið sem notandi skannaði í farsímaforriti vöruhúsakerfis var bæði gild staðsetning og gilt númerakenni. Hins vegar var þetta vandamál leyst í útgáfu 10.0.11.
