# <a name="view-an-expense-report"></a><span data-ttu-id="b0921-103">Skoða kostnaðarskýrslu</span><span class="sxs-lookup"><span data-stu-id="b0921-103">View an expense report</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="b0921-104">Þegar keyrð er fyrirspurn um að skoða færslur í fjárhag eða innkaupum og aðföngum, getur fyrirspurnin skilað of miklu upplýsingum.</span><span class="sxs-lookup"><span data-stu-id="b0921-104">When you run a query to view transactions in General ledger or Procurement and sourcing, the query might return too much information.</span></span> <span data-ttu-id="b0921-105">Þú gætir því prófað að þrengja leitina.</span><span class="sxs-lookup"><span data-stu-id="b0921-105">Therefore, you might want to narrow your search.</span></span> <span data-ttu-id="b0921-106">Til að fá þær upplýsingar sem þú þarfnast getur þú kafa niður í færslu til að sjá kostnaðarskýrsluna sem hún birtist á.</span><span class="sxs-lookup"><span data-stu-id="b0921-106">To get the information that you require, you can drill down into a transaction to see the expense report that it appeared on.</span></span>

<span data-ttu-id="b0921-107">Fyrir kostnaðarskýrslur er hægt að kafa niður frá færslu til upprunalegu skýrslunnar sem er geymd í útgjaldastýringu.</span><span class="sxs-lookup"><span data-stu-id="b0921-107">For expense reports, you can drill down from a transaction to the original report that is stored in Expense management.</span></span> <span data-ttu-id="b0921-108">Til viðbótar við heildarupphæð kostnaðar er hægt að skoða sundurliðaðar færslur, færslur sem eru gjaldgengar fyrir endurheimt á virðisaukaskatti (VSK) og kvittanir sem fylgja kostnaði.</span><span class="sxs-lookup"><span data-stu-id="b0921-108">In addition to the total amount of expenses, you can view the itemized transactions, the transactions that are eligible for recovery of value-added tax (VAT), and the receipts that are attached to the expenses.</span></span>

<span data-ttu-id="b0921-109">Til að kafa niður í upprunalega kostnaðarskýrslu sem færsla kemur fyrir á skal velja færsluna og svo velja **Upprunalegt skjal**.</span><span class="sxs-lookup"><span data-stu-id="b0921-109">To drill down to the original expense report that a transaction appeared on, select the transaction, and then select **Original document**.</span></span>

<span data-ttu-id="b0921-110">Þegar einstaka færslur eru skoðaðar er einnig hægt að skoða önnur viðbótargögn sem fylgja færslu.</span><span class="sxs-lookup"><span data-stu-id="b0921-110">When you view individual transactions, you can also view any additional data that is attached to a transaction.</span></span> <span data-ttu-id="b0921-111">Til dæmis er hægt fyrir sundurliðaðan hótelkostnað að skoða ítarlegan lista yfir kostnað eins og netaðgang, símtöl og herbergisþjónustu.</span><span class="sxs-lookup"><span data-stu-id="b0921-111">For example, for an itemized hotel expense, you can view a detailed list of expenses such as internet access, telephone calls, and room service.</span></span>
