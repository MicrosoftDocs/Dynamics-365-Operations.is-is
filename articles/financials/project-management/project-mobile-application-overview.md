# <a name="project-mobile-applications-overview"></a>Yfirlit yfir farsímaforrit verks

[!include [banner](../includes/banner.md)]

Þrjú farsímaforrit sem tengjast verktíma eru í boði fyrir Microsoft Dynamics 365: Microsoft Dynamics 365 Project Timesheet, Vinnustundafærsla verks og Vinnukortin mín/Vinnukort fyrir endurskoðun (fínstillt fyrir fartæki).

## <a name="project-timesheet-mobile-app"></a>Farsímaforritið Vinnustundafærsla verks

Farsímaforritið Dynamics 365 Project Timesheet leyfir notendum að senda inn og samþykkja vinnukort fyrir verk í fartækjum þeirra. Þetta farsímaforrit birtir virkni vinnukorts í verkefnastjórnunar- og bókhaldssvæði fyrir Microsoft Dynamics 365 for Finance and Operations. Það auðveldar afkastagetu og skilvirkni notanda og hjálpar einnig með vinnustundafærslur og samþykki á vinnukortum verks.

## <a name="project-time-entry-workspace"></a>Vinnusvæði Project Time Entry

Vinnusvæði Project Time Entry gerir notendum kleift að færa inn og vista tíma fyrir verk. Það leyfir notendum hins vegar ekki að senda inn eða samþykkja vinnukort verks. Einungis er hægt að nota það fyrir vinnustundafærslu. Þetta fartækjavinnusvæði er ætlað til að nota saman með Microsoft Dynamics 365 fyrir forritið Unified Operations Mobile.

## <a name="my-timesheetstimesheets-for-my-review"></a>Vinnukortin mín/Vinnukort fyrir endurskoðun

Vinnukortin mín (fínstillt fyrir fartæki) og Vinnukort fyrir endurskoðun (fínstillt fyrir fartæki) er lausn sem gerir starfsmönnum kleift að stofna, senda inn og samþykkja vinnukort verks í fartæki. Starfskraftar hafa aðgang að virkni Finance and Operations í gegnum vefslóð forritsins. Þessi lausn krefst ekki Dynamics 365 fyrir forritið Unified Operations Mobile.

Frekari upplýsingar er að finna í [Microsoft Dynamics 365 Project Timesheet farsímaforritið](project-timesheet.md), [Vinnustundafærsla verks]( project-time-entry-mobile-workspace.md) og [Project Timesheets í fartæki](Mobile-timesheets.md).
