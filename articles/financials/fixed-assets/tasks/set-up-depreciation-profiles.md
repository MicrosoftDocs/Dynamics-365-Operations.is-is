# <a name="set-up-and-create-depreciation-profiles"></a>Setja upp og búa til afskriftarreglur

[!include [task guide banner](../../includes/task-guide-banner.md)]

Afskriftareglur ákvarða tegund og tíðni afskriftar fyrir eigna.   Þessi skráning notar USMF sýnigögn fyrirtækis og hlutverk bókhaldara.


## <a name="create-a-depreciation-profile"></a>Stofna afskriftarreglu
1. Opna Eignir > Uppsetning > Afskriftarreglur.
2. Smellið á „Nýtt“.
3. Færa inn gildi í svæðinu afskriftarregla.
4. Í reitinn Heiti skal slá inn gildi.
5. Veljið valkost í svæðinu aðferð.
    * Ef valið er minnka stöðu, þarf að færa inn prósentu í reitinn Prósentu.  
6. Í reitnum afskriftarár skal velja valkost.
7. Í reitnum tímabilstíðni skal velja valkost.
8. Lokaðu skjámyndinni.
