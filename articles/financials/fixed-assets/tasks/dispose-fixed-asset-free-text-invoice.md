# <a name="dispose-of-a-fixed-asset-using-a-free-text-invoice"></a><span data-ttu-id="b54e9-103">Selja eign með reikningi með frjálsum texta</span><span class="sxs-lookup"><span data-stu-id="b54e9-103">Dispose of a fixed asset using a free text invoice</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="b54e9-104">Þetta ferli sýnir hvernig losnað er við eign með reikningi með frjálsum texta.</span><span class="sxs-lookup"><span data-stu-id="b54e9-104">This procedure shows how to dispose of a fixed asset using the free text invoice.</span></span>

1. <span data-ttu-id="b54e9-105">Fara í Viðskiptakröfur > Reikningar > Allir reikningar með frjálsum texta.</span><span class="sxs-lookup"><span data-stu-id="b54e9-105">Go to Accounts receivable > Invoices > All free text invoices.</span></span>
2. <span data-ttu-id="b54e9-106">Smellið á „Nýtt“.</span><span class="sxs-lookup"><span data-stu-id="b54e9-106">Click New.</span></span>
3. <span data-ttu-id="b54e9-107">Færa inn eða veljið gildi í svæðinu Reikningur viðskiptavinar.</span><span class="sxs-lookup"><span data-stu-id="b54e9-107">In the Customer account field, enter or select a value.</span></span>
4. <span data-ttu-id="b54e9-108">Staðfesta sjálfgefna reikningsdagsetningu og breyta ef við á.</span><span class="sxs-lookup"><span data-stu-id="b54e9-108">Validate the default Invoice date and edit if applicable.</span></span>
5. <span data-ttu-id="b54e9-109">Staðfesta útistandandi sjálfgefna hausareiti á borð við Gjaldmiðil og breytið ef við á.</span><span class="sxs-lookup"><span data-stu-id="b54e9-109">Validate remaining default header fields, such as Currency and edit if applicable.</span></span>
6. <span data-ttu-id="b54e9-110">Slá inn lýsingu á reikningslínu.</span><span class="sxs-lookup"><span data-stu-id="b54e9-110">Enter a Description into the invoice line.</span></span>
7. <span data-ttu-id="b54e9-111">Slá inn eða velja aðallykil fyrir reikningslínu.</span><span class="sxs-lookup"><span data-stu-id="b54e9-111">Enter or select the Main account for the invoice line.</span></span>
8. <span data-ttu-id="b54e9-112">Staðfesta sjálfgefna VSK-flokka og VSK-flokka fyrir vörur.</span><span class="sxs-lookup"><span data-stu-id="b54e9-112">Validate the default Sales tax group and Item sales tax group.</span></span>
9. <span data-ttu-id="b54e9-113">Slá inn einingarverð eða upphæð sölu eignarinnar.</span><span class="sxs-lookup"><span data-stu-id="b54e9-113">Enter the Unit price or hte Amount of the sale of the fixed asset.</span></span>
10. <span data-ttu-id="b54e9-114">Smella á línuupplýsingar.</span><span class="sxs-lookup"><span data-stu-id="b54e9-114">Click Line details.</span></span>  
11. <span data-ttu-id="b54e9-115">Velja númer eignar til að selja.</span><span class="sxs-lookup"><span data-stu-id="b54e9-115">Select the Fixed asset number to be sold.</span></span>
12. <span data-ttu-id="b54e9-116">Smellið á „Bóka“.</span><span class="sxs-lookup"><span data-stu-id="b54e9-116">Click Post.</span></span>
