# <a name="list-of-er-functions-in-the-record-category"></a>Listi yfir ER-aðgerðir í skráaflokknum

[!include [banner](../includes/banner.md)]

Hægt er að nota skráaraðgerðir rafrænnar skýrslugerðar (ER) til að draga upplýsingar úr og framkvæma aðgerðir á gagnagjafa gögnin *Ílát (skrá)*. Þetta efni gefur yfirlit yfir þessar aðgerðir.

## <a name="list-of-supported-functions"></a>Listi yfir studdar aðgerðir

| Aðgerð | Lýsing |
|----------|-------------|
| [NullContainer](er-functions-record-nullcontainer.md) | Þessi aðgerð skilar núll gildi fyrir *Ílát (skrá)* sem hefur sama skipulag og tilgreindur skráalisti eða skrá. |
| [EmptyRecord](er-functions-record-emptyrecord.md) | Þessi aðgerð skilar núll gildi fyrir *Ílát (skrá)* sem hefur sama skipulag og tilgreindur skráalisti eða skrá. |
