# <a name="file-destination"></a>Viðtökustaður skráar

[!include [banner](../includes/banner.md)]

Hægt er að stilla skrá áfangastaðar fyrir hvern FOLDER eða FILE hluti á rafrænu skýrslugerðarsniðinu (ER) sem er stillt til að búa til útgönguskjöl. Byggt á stillingu ákvörðunarstaðar er myndað skjal hægt að hlaða niður úr vafranum.

Á síðunni **Stillingar áfangastaðar**, ef þú stilltir **Virkjað** á **Já**, opnast opinn eða vistaður gluggi þegar keyrslu stillingarinnar lýkur.
