# <a name="LOWER">LOWER ER-aðgerð</a>

[!include [banner](../includes/banner.md)]

Aðgerð `LOWER` skilar tilgreindum textastreng sem *Strengja*-gildi eftir að því hefur verið breytt í lágstafi.

## <a name="syntax"></a>Málskipun

```
LOWER (text)
```

## <a name="arguments"></a>Frumbreytur

`text`: *Strengur*

Gildið *Strengur* sem tilgreinir textann.

## <a name="return-values"></a>Skilagildi

*Strengur*

Textagildið sem verður til.

## <a name="example"></a>Dæmi

`LOWER ("Sample")` skilar **"sample"**.
