# <a name="UPPER">UPPER ER-aðgerð</a>

[!include [banner](../includes/banner.md)]

Aðgerðin `UPPER` skilar tilgreindum textastreng sem *Strengja*-gildi eftir að því hefur verið breytt í hástafi.

## <a name="syntax"></a>Málskipun

```vb
UPPER (text )
```

## <a name="arguments"></a>Frumbreytur

`text`: *Strengur*

Gild slóð í gagnagjafa af gerðinni *Strengur*.

## <a name="return-values"></a>Skilagildi

*Strengur*

Textagildið sem verður til.

## <a name="example"></a>Dæmi

`UPPER ("Sample")` skilar **"SAMPLE"**.
