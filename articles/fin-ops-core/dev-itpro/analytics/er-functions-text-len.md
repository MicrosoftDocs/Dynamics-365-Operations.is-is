# <a name="LEN">LEN ER aðgerð</a>

[!include [banner](../includes/banner.md)]

Aðgerðin `LEN` skilar heiltölugildi sem sýnir fjölda staftákna í tilteknum streng sem *heiltölu*-gildi.

## <a name="syntax"></a>Málskipun

```
LEN (text)
```

## <a name="arguments"></a>Frumbreytur

`text`: *Strengur*

Gildið *Strengur* sem tilgreinir textann.

## <a name="return-values"></a>Skilagildi

*Heiltala*

Tölugildið sem verður til.

## <a name="example"></a>Dæmi

`LEN ("Sample")` skilar **6**.
