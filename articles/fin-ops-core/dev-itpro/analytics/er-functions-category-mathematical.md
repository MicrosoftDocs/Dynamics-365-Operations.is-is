# <a name="list-of-er-functions-in-the-mathematical-category"></a>Listi yfir ER-aðgerðir í stærðfræðiflokknum

[!include [banner](../includes/banner.md)]

Hægt er að nota stærðfræðilegar aðgerðir rafrænnar skýrslur (ER) til að gera marga algenga stærðfræðiútreikninga. Þetta efni gefur yfirlit yfir þessar aðgerðir.

## <a name="list-of-supported-functions"></a>Listi yfir studdar aðgerðir

| Aðgerð | Lýsing |
|----------|-------------|
| [Abs](er-functions-mathematical-abs.md)             | Þessi aðgerð skilar algeru gildi (leifastofn) tiltekins fjölda sem *Raun*-gildi. Með öðrum orðum skilar hún tölunni án táknsins. |
| [Styrkur](er-functions-mathematical-power.md)         | Þessi aðgerð skilar *Raun*-gildi sem táknar niðurstöðuna af því að hækka tilgreinda jákvæða tölu upp að tilgreindri valheimild. |
| [Round](er-functions-mathematical-round.md)         | Þessi virkni skilar tilgreindri tölu sem *Raun*-gildi eftir að hún hefur verið námunduð að tilgreindum fjölda aukastafa. |
| [RoundDown](er-functions-mathematical-rounddown.md) | Þessi virkni skilar tilgreindri tölu sem *Raun*-gildi eftir að hún hefur verið námunduð niður að tilgreindum fjölda aukastafa. |
| [RoundUp](er-functions-mathematical-roundup.md)     | Þessi virkni skilar tilgreindri tölu sem *Raun*-gildi eftir að hún hefur verið námunduð upp að tilgreindum fjölda aukastafa. |
