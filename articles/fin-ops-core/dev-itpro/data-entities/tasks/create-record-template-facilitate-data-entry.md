# <a name="create-a-record-template-to-facilitate-data-entry"></a><span data-ttu-id="99346-103">Búa til færslusniðmát til að greiða fyrir skráningu gagna</span><span class="sxs-lookup"><span data-stu-id="99346-103">Create a record template to facilitate data entry</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="99346-104">Þetta efni sýnir hvernig stofna færslusniðmát til að gildi í svæði sem oft notað ekki þurfa að slá inn sérstaklega fyrir hverja ný færsla.</span><span class="sxs-lookup"><span data-stu-id="99346-104">This topic demonstrates how to create a record template so that field values that are used often do not have to be entered explicitly for each new record.</span></span> <span data-ttu-id="99346-105">Í þessu ferli er stofnuð ný færsla fyrir nýjar fartölvur sem ætti að bæta við eignir þínar.</span><span class="sxs-lookup"><span data-stu-id="99346-105">In this procedure, you'll create a new record for new laptops that should be added to your fixed assets.</span></span> <span data-ttu-id="99346-106">Þetta ferli notar USMF sýnifyrirtæki.</span><span class="sxs-lookup"><span data-stu-id="99346-106">This procedure uses the USMF sample company.</span></span>

1. <span data-ttu-id="99346-107">Í skoðunarrúðnni ferðu í **Kerfseiningar > Fastafjármunir > Fastafjármunir > Fastafjármunir**.</span><span class="sxs-lookup"><span data-stu-id="99346-107">In the navigation pane, go to **Modules > Fixed assets > Fixed assets > Fixed assets**.</span></span>
2. <span data-ttu-id="99346-108">Veljið **Nýtt**.</span><span class="sxs-lookup"><span data-stu-id="99346-108">Select **New**.</span></span>
3. <span data-ttu-id="99346-109">Í reitnum **Eignaflokkur** skal færa inn eða velja gildi.</span><span class="sxs-lookup"><span data-stu-id="99346-109">In the **Fixed asset group** field, enter or select a value.</span></span>
4. <span data-ttu-id="99346-110">Í reitinn **Heiti** skal slá inn gildi.</span><span class="sxs-lookup"><span data-stu-id="99346-110">In the **Name** field, type a value.</span></span> <span data-ttu-id="99346-111">Sláðu til dæmis inn **Fartölva yfirmanns**.</span><span class="sxs-lookup"><span data-stu-id="99346-111">For example, enter **Corporate lead laptop**.</span></span>  
5. <span data-ttu-id="99346-112">Í svæði **Leita að nafni** skal slá inn gildi.</span><span class="sxs-lookup"><span data-stu-id="99346-112">In the **Search name** field, type a value.</span></span> <span data-ttu-id="99346-113">Sláðu til dæmis inn **fartölva**.</span><span class="sxs-lookup"><span data-stu-id="99346-113">For example, enter **laptop**.</span></span>  
6. <span data-ttu-id="99346-114">Víkkaðu út hlutann **Tæknilegar upplýsingar**.</span><span class="sxs-lookup"><span data-stu-id="99346-114">Expand the **Technical information** section.</span></span>
7. <span data-ttu-id="99346-115">Í reitunum **Gerð**, **Líkan** og **Árgerð** skal slá inn gildi.</span><span class="sxs-lookup"><span data-stu-id="99346-115">In the **Make**, **Model**, and **Model year** fields, type values.</span></span>
8. <span data-ttu-id="99346-116">Í aðgerðasvæðinu velurðu **Valkostir**.</span><span class="sxs-lookup"><span data-stu-id="99346-116">On the Action Pane, select **Options**.</span></span>
9. <span data-ttu-id="99346-117">Veldu **Skrá upplýsingar**.</span><span class="sxs-lookup"><span data-stu-id="99346-117">Select **Record info**.</span></span>
10. <span data-ttu-id="99346-118">Velja **Notandasniðmát**.</span><span class="sxs-lookup"><span data-stu-id="99346-118">Select **User template**.</span></span>
11. <span data-ttu-id="99346-119">Í reitinn **Heiti** skal slá inn gildi.</span><span class="sxs-lookup"><span data-stu-id="99346-119">In the **Name** field, type a value.</span></span>
12. <span data-ttu-id="99346-120">Í reitinn **Lýsing** skal slá inn gildi.</span><span class="sxs-lookup"><span data-stu-id="99346-120">In the **Description** field, type a value.</span></span>
13. <span data-ttu-id="99346-121">Veljið **Í lagi**.</span><span class="sxs-lookup"><span data-stu-id="99346-121">Select **OK**.</span></span>
14. <span data-ttu-id="99346-122">Veljið **Loka**.</span><span class="sxs-lookup"><span data-stu-id="99346-122">Select **Close**.</span></span>
