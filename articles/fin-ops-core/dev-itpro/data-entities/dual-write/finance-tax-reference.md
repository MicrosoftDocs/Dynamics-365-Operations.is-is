# <a name="access-to-finance-and-tax-reference-data"></a>Aðgangur að fjárhags- og skattatilvísunargögnum

[!include [banner](../../includes/banner.md)]



Sérhver viðskipti vinnur með grunnupplýsingar af fjárhagslegum gögnum, svo sem reikningsárinu, gjaldmiðlinum sem viðskipti eru í, reikninga sem peningarnir til að reka fyrirtækið koma til eða fara út úr, skatthlutföll og endurgreiðsla. Þessi gögn eru í forritum Finance and Operations. Hins vegar er það útsett fyrir Common Data Service svo að líkanadrifin forrit í Microsoft Dynamics 365 geta verið með eina heimild til fjár- og skattaupplýsinga. Á þennan hátt eru gögn samræmd yfir vistkerfi fyrirtækisins. 

Fjárhags- og skattagögn eru samþætt með eftirfarandi vörpunum:

+ [Samþættur fjárhagur](ledger-mapping.md)
+ [Samþætt aðalgögn skatta](tax-mapping.md)
