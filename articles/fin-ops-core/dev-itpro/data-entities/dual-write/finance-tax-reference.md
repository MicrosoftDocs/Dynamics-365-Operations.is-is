# <a name="access-to-finance-and-tax-reference-data"></a>Aðgangur að fjárhags- og skattatilvísunargögnum

[!include [banner](../../includes/banner.md)]



Sérhver viðskipti vinnur með grunnupplýsingar af fjárhagslegum gögnum, svo sem reikningsárinu, gjaldmiðlinum sem viðskipti eru í, reikninga sem peningarnir til að reka fyrirtækið koma til eða fara út úr, skatthlutföll og endurgreiðsla. Þessi gögn eru í Finance and Operations öppum. Hinsvegar eru þau sýnd í Dataverse þannig að forrit viðskiptavinar geta verið með einn uppruna fyrir fjármála- og skattagögn. Á þennan hátt eru gögn samræmd yfir vistkerfi fyrirtækisins.

Fjárhags- og skattagögn eru samþætt með eftirfarandi vörpunum:

+ [Samþættur fjárhagur](ledger-mapping.md)
+ [Samþætt aðalgögn skatta](tax-mapping.md)
