# <a name="change-the-banner-or-logo"></a><span data-ttu-id="8a37f-103">Breyta borða eða kennimerki</span><span class="sxs-lookup"><span data-stu-id="8a37f-103">Change the banner or logo</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="8a37f-104">Eftirfarandi ferli er listi yfir skref sem kerfisstjórum má nota til að uppfæra borða eða merki fyrirtækis sem er birt fyrir lögaðila.</span><span class="sxs-lookup"><span data-stu-id="8a37f-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="8a37f-105">Sýnigögn fyrirtækisins til að stofna þetta ferli er USMF.</span><span class="sxs-lookup"><span data-stu-id="8a37f-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="8a37f-106">Farðu í **Skoðunarrúðu > Kerfiseiningar > Fyrirtækisstjórnun > Fyrirtæki > Lögaðilar**.</span><span class="sxs-lookup"><span data-stu-id="8a37f-106">Go to **Navigation pane > Modules > Organization administration > Organizations > Legal entities**.</span></span>
2. <span data-ttu-id="8a37f-107">Velja lögaðilann sem á að uppfæra borða eða merki fyrirtækis af listanum vinstra megin.</span><span class="sxs-lookup"><span data-stu-id="8a37f-107">In the list on the left, select the legal entity for which you want to update the banner or logo.</span></span> <span data-ttu-id="8a37f-108">Ef hann er þegar valinn fara í næsta skref.</span><span class="sxs-lookup"><span data-stu-id="8a37f-108">If it is already selected, go to the next step.</span></span>
3. <span data-ttu-id="8a37f-109">Stækkaðu flipann **Mælaborðsmynd**.</span><span class="sxs-lookup"><span data-stu-id="8a37f-109">Expand **Dashboard image** tab.</span></span>
4. <span data-ttu-id="8a37f-110">Veldu **Breyta**.</span><span class="sxs-lookup"><span data-stu-id="8a37f-110">Select **Change**.</span></span>
    
    <span data-ttu-id="8a37f-111">Tilvalin upplausn fyrir borðamynd er 1920 x 281 dílar.</span><span class="sxs-lookup"><span data-stu-id="8a37f-111">The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="8a37f-112">Breidd tilvalið fyrir mynd með fyrirtækismerki er 350 dílar.</span><span class="sxs-lookup"><span data-stu-id="8a37f-112">The ideal width for a logo image is 350 pixels.</span></span>
    
5. <span data-ttu-id="8a37f-113">Veljið **Vista**.</span><span class="sxs-lookup"><span data-stu-id="8a37f-113">Select **Save**.</span></span>
6. <span data-ttu-id="8a37f-114">Farðu í **Skoðunarrúðu > Kerfiseiningar > Almennt > Almennt > Sjálfgefið mælaborð**.</span><span class="sxs-lookup"><span data-stu-id="8a37f-114">Go to **Navigation pane > Modules > Common > Common > Default dashboard**.</span></span> <span data-ttu-id="8a37f-115">Ný borði eða merki fyrirtækis ættu að birtast á á yfirliti.</span><span class="sxs-lookup"><span data-stu-id="8a37f-115">You should see the new banner or logo image on the dashboard.</span></span>  
    
    <span data-ttu-id="8a37f-116">Hugsanlega þarf að breyta fyrirtækinu, og nota fyrirtækjaval, á það sem borða var hlaðið upp fyrir.</span><span class="sxs-lookup"><span data-stu-id="8a37f-116">You may need to change your company, using the company picker, to the one you uploaded the banner for.</span></span>
