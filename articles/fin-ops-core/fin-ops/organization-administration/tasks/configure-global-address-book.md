# <a name="configure-the-global-address-book"></a><span data-ttu-id="8250d-103">Grunnstilla altæka aðsetursbók</span><span class="sxs-lookup"><span data-stu-id="8250d-103">Configure the global address book</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="8250d-104">Notið þetta ferli til að setja upp sjálfgildi og öryggisreglur fyrir altæka aðsetursbók.</span><span class="sxs-lookup"><span data-stu-id="8250d-104">Use this procedure to set the default values and security policies for the global address book.</span></span> 

<span data-ttu-id="8250d-105">Sýnigögn gögn fyrirtækisins til að stofna verkið er USMF.</span><span class="sxs-lookup"><span data-stu-id="8250d-105">The demo data company used to create this task is USMF.</span></span> <span data-ttu-id="8250d-106">Þetta verk er ætlað fyrir hópinn Áætlanagerð og afbrigði.</span><span class="sxs-lookup"><span data-stu-id="8250d-106">This task is intended for the Planning and configuration team.</span></span>

1. <span data-ttu-id="8250d-107">Í skoðunarrúðunni ferðu í **Kerfi > Fyrirtækisstjórnun > Altæk aðsetursbók > Færibreytur altækrar aðsetursbókar**.</span><span class="sxs-lookup"><span data-stu-id="8250d-107">In the Navigation pane, go to **Modules > Organization administration > Global address book > Global address book parameters**.</span></span>
2. <span data-ttu-id="8250d-108">Í reitnum **Nafnaröð** velurðu hvernig á að sýna nöfn.</span><span class="sxs-lookup"><span data-stu-id="8250d-108">In the **Name sequence** field, select how names should be shown.</span></span>
3. <span data-ttu-id="8250d-109">Í **Eyða aðilum án hlutverka** velurðu hvort eyða eigi aðilum sem hafa ekki fengið hlutverk.</span><span class="sxs-lookup"><span data-stu-id="8250d-109">In **Delete parties with no roles**, select whether to delete parties with that have not been assigned a role.</span></span>
4. <span data-ttu-id="8250d-110">Í **Nota tvítekningaprófun** velurðu hvort leitað skuli að tvíteknum færslum.</span><span class="sxs-lookup"><span data-stu-id="8250d-110">In **Use duplicate check**, select whether to check for duplicate records.</span></span>
5. <span data-ttu-id="8250d-111">Í **Birta DUNS-númer á heimilisföngum** velurðu hvort birta eigi DUNS-númerið á heimilisföngum.</span><span class="sxs-lookup"><span data-stu-id="8250d-111">In **Display DUNS number on addresses**, select whether to display the DUNS number on addresses.</span></span>
6. <span data-ttu-id="8250d-112">Í **Athuga einkvæmt DUNS-númer** velurðu hvort leita eigi að einstökum DUNS-númerum.</span><span class="sxs-lookup"><span data-stu-id="8250d-112">In **Check for unique DUNS number**, select whether to check for unique DUNS numbers.</span></span>
7. <span data-ttu-id="8250d-113">Í reitnum **Aðili** skal velja valkost.</span><span class="sxs-lookup"><span data-stu-id="8250d-113">In the **Party** field, select an option.</span></span>
8. <span data-ttu-id="8250d-114">Veldu valkost í reitnum **Viðskiptavinur**.</span><span class="sxs-lookup"><span data-stu-id="8250d-114">In the **Customer** field, select an option.</span></span>
9. <span data-ttu-id="8250d-115">Veldu valkost í reitnum **Lánardrottinn**.</span><span class="sxs-lookup"><span data-stu-id="8250d-115">In the **Vendor** field, select an option.</span></span>
10. <span data-ttu-id="8250d-116">Veldu valkost í reitnum **Viðfang**.</span><span class="sxs-lookup"><span data-stu-id="8250d-116">In the **Prospect** field, select an option.</span></span>
11. <span data-ttu-id="8250d-117">Í reitnum **Samkeppnisaðili** skal velja valkost.</span><span class="sxs-lookup"><span data-stu-id="8250d-117">In the **Competitor** field, select an option.</span></span>
12. <span data-ttu-id="8250d-118">Smelltu á flipann **Öryggi einkastaðsetningar**.</span><span class="sxs-lookup"><span data-stu-id="8250d-118">Click the **Private location security** tab.</span></span>
13. <span data-ttu-id="8250d-119">Í listanum skal finna og velja þá skráningu sem óskað er eftir.</span><span class="sxs-lookup"><span data-stu-id="8250d-119">In the list, find and select the desired record.</span></span> <span data-ttu-id="8250d-120">Styðja á Shift-takkanum til að velja mörg hlutverk til að bæta við reitinn **Valin hlutverk** og smellið á örina til að bæta við völdum hlutverkum.</span><span class="sxs-lookup"><span data-stu-id="8250d-120">Press the Shift key to select multiple roles to add to the **Selected roles** pane and then click the arrow to add the selected roles.</span></span>  
14. <span data-ttu-id="8250d-121">Smellt er á **Vista**.</span><span class="sxs-lookup"><span data-stu-id="8250d-121">Click **Save**.</span></span>
