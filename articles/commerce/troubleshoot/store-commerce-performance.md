# <a name="troubleshoot-store-commerce-performance-issues"></a>Leysa vandamál í afkomu verslana

[!include [banner](../includes/banner.md)]

Þessi grein útskýrir hvernig á að leysa afköst vandamál í Microsoft Dynamics 365 Commerce Store Commerce app.

- Þegar þú leysa vandamál afkasta fyrir Store Commerce, forðast fjarinnskráningu. Í staðinn skaltu skrá þig beint inn á Store Commerce tækið.
- Ef Store Commerce appið er hægt, athugaðu örgjörva, minni, netkerfi og diskanotkun með því að nota Task Manager eða Resource Monitor. Gakktu úr skugga um að engin veruleg neysla sé á auðlindum tækisins.
- Athugaðu nettengingarstöðu þína og staðfestu að HTTPS beiðnir/svör taki ekki meira en nokkrar sekúndur.
