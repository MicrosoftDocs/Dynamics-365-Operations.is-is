<span data-ttu-id="2351f-106">Margmiðlunarsafn vefsvæðishönnuðar Commerce gerir þér kleift að klippa myndir til að fínstilla þær fyrir mismunandi gerðir eininga og skoðunargáttir.</span><span class="sxs-lookup"><span data-stu-id="2351f-106">The Commerce site builder Media Library allows you to crop images to optimize them for different module types and viewports.</span></span>

## <a name="crop-an-image"></a><span data-ttu-id="2351f-107">Skera mynd</span><span class="sxs-lookup"><span data-stu-id="2351f-107">Crop an image</span></span>

<span data-ttu-id="2351f-108">Fylgdu þessum skrefum til að skera mynd í vefsvæðishönnuði.</span><span class="sxs-lookup"><span data-stu-id="2351f-108">To crop an image in site builder, follow these steps.</span></span>

1. <span data-ttu-id="2351f-109">Í vinstri flettiglugganum á vefsvæðishönnuði Commerce velurðu **Margmiðlunarsafn**.</span><span class="sxs-lookup"><span data-stu-id="2351f-109">In the left navigation pane of Commerce site builder, select **Media Library**.</span></span>
1. <span data-ttu-id="2351f-110">Í aðalglugganum velurðu myndina sem þú vilt breyta.</span><span class="sxs-lookup"><span data-stu-id="2351f-110">In the main window, select the image you want to modify.</span></span>
1. <span data-ttu-id="2351f-111">Á skipanastikunni velurðu **Breyta**.</span><span class="sxs-lookup"><span data-stu-id="2351f-111">On the command bar, select **Edit**.</span></span>
1. <span data-ttu-id="2351f-112">Veldu myndina sem á að fara inn í **Breyta stillingu**.</span><span class="sxs-lookup"><span data-stu-id="2351f-112">Select the image to enter **Edit Mode**.</span></span>
1. <span data-ttu-id="2351f-113">Undir **Breyta stillingu** velurðu **Breyta sýn eftir einingu**.</span><span class="sxs-lookup"><span data-stu-id="2351f-113">Under **Edit Mode**, select **Edit View by Module**.</span></span>
1. <span data-ttu-id="2351f-114">Úr fellivalmyndinni **Einingu** velurðu gerð einingar.</span><span class="sxs-lookup"><span data-stu-id="2351f-114">From the **Module** drop-down menu, select the module type.</span></span>
1. <span data-ttu-id="2351f-115">Úr fellivalmyndinni **Gerð sýnar** velurðu gerð sýnar.</span><span class="sxs-lookup"><span data-stu-id="2351f-115">From the **View type** drop-down menu, select the view type.</span></span>
1. <span data-ttu-id="2351f-116">Úr fellilistanum **Staðsetning** velurðu staðsetningu myndar.</span><span class="sxs-lookup"><span data-stu-id="2351f-116">From the **Placement** drop-down menu, select the image placement.</span></span>
1. <span data-ttu-id="2351f-117">Úr fellivalmyndinni **Skoðunargátt** velurðu stærð skoðunargáttar.</span><span class="sxs-lookup"><span data-stu-id="2351f-117">From the **Viewport** drop-down menu, select the viewport size.</span></span>
1. <span data-ttu-id="2351f-118">Myndin er lögð yfir svæðið sem táknar skurðsvæðið.</span><span class="sxs-lookup"><span data-stu-id="2351f-118">The image is overlaid with the area representing the crop region.</span></span> <span data-ttu-id="2351f-119">Færðu og breyttu skurðsvæðinu eftir þörfum.</span><span class="sxs-lookup"><span data-stu-id="2351f-119">Move and resize the crop region as needed.</span></span> <span data-ttu-id="2351f-120">Þáttarhlutfallinu verður viðhaldið sjálfkrafa.</span><span class="sxs-lookup"><span data-stu-id="2351f-120">The aspect ratio will be maintained automatically.</span></span>
1. <span data-ttu-id="2351f-121">Þegar það er búið skaltu á skipanastikunni velja **Vista** og veldu síðan **Ljúka breytingu**.</span><span class="sxs-lookup"><span data-stu-id="2351f-121">When you're done, on the command bar, select **Save**, and then select **Finish editing**.</span></span> 

<span data-ttu-id="2351f-122">Eftir að sérsniðnum skurði er lokið taka myndbreytingar gildi næstum því strax.</span><span class="sxs-lookup"><span data-stu-id="2351f-122">After custom cropping is completed, image modifications will take effect almost immediately.</span></span>
