## <a name="add-a-copyright-notice"></a><span data-ttu-id="9b259-109">Bæta við yfirlýsingu um höfundarrétt</span><span class="sxs-lookup"><span data-stu-id="9b259-109">Add a copyright notice</span></span>

<span data-ttu-id="9b259-110">Fylgdu þessum skrefum til að bæta við höfundarréttartilkynningu neðst á hverri síðu sem notar ákveðið sniðmát.</span><span class="sxs-lookup"><span data-stu-id="9b259-110">To add a copyright notice to the bottom of every page that uses a specific template, follow these steps.</span></span>

1. <span data-ttu-id="9b259-111">Farðu í **Brot** og veldu síðan **Nýtt síðubrot**.</span><span class="sxs-lookup"><span data-stu-id="9b259-111">Go to **Fragments**, and then select **New Page Fragment**.</span></span>
1. <span data-ttu-id="9b259-112">Í glugganum velurðu eininguna **Síðufótur** og gefur brotinu heiti.</span><span class="sxs-lookup"><span data-stu-id="9b259-112">In the dialog box, select the **Footer** module, and name the fragment.</span></span> <span data-ttu-id="9b259-113">Til dæmis, sláðu inn **Footer-Copyright**.</span><span class="sxs-lookup"><span data-stu-id="9b259-113">For example, enter **Footer-Copyright**.</span></span>
1. <span data-ttu-id="9b259-114">Veljið **Í lagi**.</span><span class="sxs-lookup"><span data-stu-id="9b259-114">Select **OK**.</span></span>
1. <span data-ttu-id="9b259-115">Í glugganum velurðu úrfellingarhnappinn (**...**) við hliðina á **síðufæti** og velur síðan **Bæta við einingu**.</span><span class="sxs-lookup"><span data-stu-id="9b259-115">In the navigation pane, select the ellipsis button (**...**) next to **Footer**, and then select **Add Module**.</span></span>
1. <span data-ttu-id="9b259-116">Í valmyndinni velurðu **Síðufótaflokkur** og velur síðan **Í lagi**.</span><span class="sxs-lookup"><span data-stu-id="9b259-116">In the dialog box, select **Footer category**, and then select **OK**.</span></span>
1. <span data-ttu-id="9b259-117">Í glugganum velurðu úrfellingarhnappinn við hliðina á **síðufótaflokki** og velur síðan **Bæta við einingu**.</span><span class="sxs-lookup"><span data-stu-id="9b259-117">In the navigation pane, select the ellipsis button next to **Footer category**, and then select **Add Module**.</span></span>
1. <span data-ttu-id="9b259-118">Í valmyndinni velurðu **vöru í bálki með fjölbreyttu efni** og velur síðan **Í lagi**.</span><span class="sxs-lookup"><span data-stu-id="9b259-118">In the dialog box, select **Content rich block item**, and then select **OK**.</span></span>
1. <span data-ttu-id="9b259-119">Í valmyndinni velurðu **Vöru í bálki með fjölbreyttu efni**.</span><span class="sxs-lookup"><span data-stu-id="9b259-119">In the navigation pane, select **Content rich block item**.</span></span>
1. <span data-ttu-id="9b259-120">Í eiginleikarrúðunni til hægri, í reitnum **Málsgrein**, bætirðu höfundarréttarskilaboðunum við.</span><span class="sxs-lookup"><span data-stu-id="9b259-120">In the properties pane on the right, in the **Paragraph** field, add your copyright message.</span></span> <span data-ttu-id="9b259-121">Til dæmis, færðu inn **(C) Fabrikam 2019**.</span><span class="sxs-lookup"><span data-stu-id="9b259-121">For example, enter **(C) Fabrikam 2019**.</span></span>
1. <span data-ttu-id="9b259-122">Veldu **Vista**, veldu **Innskráning** og veldu síðan **Birta**.</span><span class="sxs-lookup"><span data-stu-id="9b259-122">Select **Save**, select **Check In**, and then select **Publish**.</span></span>
1. <span data-ttu-id="9b259-123">Farðu í **Sniðmát**, veldu sniðmátið og veldu síðan **Skrá út**.</span><span class="sxs-lookup"><span data-stu-id="9b259-123">Go to **Templates**, select the template, and then select **Check Out**.</span></span>
1. <span data-ttu-id="9b259-124">Undir **Útlínur síðu** stækkarðu **Meginmál** og stækkar síðan **Sjálfgefin síða**.</span><span class="sxs-lookup"><span data-stu-id="9b259-124">Under **Page Outline**, expand **Body**, and then expand **Default Page**.</span></span>
1. <span data-ttu-id="9b259-125">Veldu úrfellingarhnappinn við hliðina **Síðufótarhólf** og velur síðan **Bæta við broti**.</span><span class="sxs-lookup"><span data-stu-id="9b259-125">Select the ellipsis button next to **Footer Slot**, and then select **Add Fragment**.</span></span>
1. <span data-ttu-id="9b259-126">Veldu brotið sem þú bjóst til áður og veldu síðan **Velja**.</span><span class="sxs-lookup"><span data-stu-id="9b259-126">Select the fragment that you created earlier, and then select **Select**.</span></span>
1. <span data-ttu-id="9b259-127">Skráðu brotið út í sniðmátinu og birtu það.</span><span class="sxs-lookup"><span data-stu-id="9b259-127">Check in the template, and publish it.</span></span>

<span data-ttu-id="9b259-128">Síðufótur sem inniheldur höfundarréttartilkynningu birtist sjálfkrafa neðst á öllum síðunum sem nota valda sniðmátið.</span><span class="sxs-lookup"><span data-stu-id="9b259-128">The footer that contains the copyright notice automatically appears at the bottom of all pages that use the selected template.</span></span>
