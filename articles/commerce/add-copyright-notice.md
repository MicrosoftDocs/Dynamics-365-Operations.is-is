## <a name="add-a-copyright-notice"></a><span data-ttu-id="4bfa1-109">Bæta við yfirlýsingu um höfundarrétt</span><span class="sxs-lookup"><span data-stu-id="4bfa1-109">Add a copyright notice</span></span>

<span data-ttu-id="4bfa1-110">Fylgdu þessum skrefum til að bæta við höfundarréttartilkynningu neðst á hverri síðu sem notar ákveðið sniðmát.</span><span class="sxs-lookup"><span data-stu-id="4bfa1-110">To add a copyright notice to the bottom of every page that uses a specific template, follow these steps.</span></span>

1. <span data-ttu-id="4bfa1-111">Farðu í **Brot** og veldu síðan **Nýtt**.</span><span class="sxs-lookup"><span data-stu-id="4bfa1-111">Go to **Fragments**, and then select **New**.</span></span>
1. <span data-ttu-id="4bfa1-112">Í svarglugganum **Nýtt brot** skal velja eininguna **Síðufótur** og gefa brotinu heiti.</span><span class="sxs-lookup"><span data-stu-id="4bfa1-112">In the **New fragment** dialog box, select the **Footer** module, and name the fragment.</span></span> <span data-ttu-id="4bfa1-113">Til dæmis, sláðu inn **Footer-Copyright**.</span><span class="sxs-lookup"><span data-stu-id="4bfa1-113">For example, enter **Footer-Copyright**.</span></span>
1. <span data-ttu-id="4bfa1-114">Veljið **Í lagi**.</span><span class="sxs-lookup"><span data-stu-id="4bfa1-114">Select **OK**.</span></span>
1. <span data-ttu-id="4bfa1-115">Í glugganum velurðu úrfellingarhnappinn (**...**) við hliðina á **síðufæti** og velur síðan **Bæta við einingu**.</span><span class="sxs-lookup"><span data-stu-id="4bfa1-115">In the navigation pane, select the ellipsis button (**...**) next to **Footer**, and then select **Add Module**.</span></span>
1. <span data-ttu-id="4bfa1-116">Í valmyndinni velurðu **Síðufótaflokkur** og velur síðan **Í lagi**.</span><span class="sxs-lookup"><span data-stu-id="4bfa1-116">In the dialog box, select **Footer category**, and then select **OK**.</span></span>
1. <span data-ttu-id="4bfa1-117">Í glugganum velurðu úrfellingarhnappinn við hliðina á **síðufótaflokki** og velur síðan **Bæta við einingu**.</span><span class="sxs-lookup"><span data-stu-id="4bfa1-117">In the navigation pane, select the ellipsis button next to **Footer category**, and then select **Add Module**.</span></span>
1. <span data-ttu-id="4bfa1-118">Í valmyndinni velurðu **Textabálkur** og velur síðan **Í lagi**.</span><span class="sxs-lookup"><span data-stu-id="4bfa1-118">In the dialog box, select **Text block**, and then select **OK**.</span></span>
1. <span data-ttu-id="4bfa1-119">Veldu á leiðsöguskjánum **Textabálkur**.</span><span class="sxs-lookup"><span data-stu-id="4bfa1-119">In the navigation pane, select **Text block**.</span></span>
1. <span data-ttu-id="4bfa1-120">Í eiginleikarrúðunni til hægri, í reitnum **Málsgrein**, bætirðu höfundarréttarskilaboðunum við.</span><span class="sxs-lookup"><span data-stu-id="4bfa1-120">In the properties pane on the right, in the **Paragraph** field, add your copyright message.</span></span> <span data-ttu-id="4bfa1-121">Til dæmis, færðu inn **(C) Fabrikam 2019**.</span><span class="sxs-lookup"><span data-stu-id="4bfa1-121">For example, enter **(C) Fabrikam 2019**.</span></span>
1. <span data-ttu-id="4bfa1-122">Veldu **Vista**, veldu **Ljúka við breytingar** og veldu síðan **Birta**.</span><span class="sxs-lookup"><span data-stu-id="4bfa1-122">Select **Save**, select **Finish editing**, and then select **Publish**.</span></span>
1. <span data-ttu-id="4bfa1-123">Opnaðu **Sniðmát**, veldu sniðmátið og veldu síðan **Breyta**.</span><span class="sxs-lookup"><span data-stu-id="4bfa1-123">Go to **Templates**, select the template, and then select **Edit**.</span></span>
1. <span data-ttu-id="4bfa1-124">Undir **Útlínur síðu** stækkarðu **Meginmál** og stækkar síðan **Sjálfgefin síða**.</span><span class="sxs-lookup"><span data-stu-id="4bfa1-124">Under **Page Outline**, expand **Body**, and then expand **Default Page**.</span></span>
1. <span data-ttu-id="4bfa1-125">Veldu úrfellingarhnappinn við hliðina **Síðufótarhólf** og velur síðan **Bæta við broti**.</span><span class="sxs-lookup"><span data-stu-id="4bfa1-125">Select the ellipsis button next to **Footer Slot**, and then select **Add Fragment**.</span></span>
1. <span data-ttu-id="4bfa1-126">Veldu brotið sem þú bjóst til áður og veldu síðan **Velja**.</span><span class="sxs-lookup"><span data-stu-id="4bfa1-126">Select the fragment that you created earlier, and then select **Select**.</span></span>
1. <span data-ttu-id="4bfa1-127">Veldu **Ljúka við breytingar** til að athuga með sniðmátið og veldu síðan **Birta** til að birta það.</span><span class="sxs-lookup"><span data-stu-id="4bfa1-127">Select **Finish editing** to check in the template, and then select **Publish** to publish it.</span></span>

<span data-ttu-id="4bfa1-128">Síðufótur sem inniheldur höfundarréttartilkynningu birtist sjálfkrafa neðst á öllum síðunum sem nota valda sniðmátið.</span><span class="sxs-lookup"><span data-stu-id="4bfa1-128">The footer that contains the copyright notice automatically appears at the bottom of all pages that use the selected template.</span></span>
