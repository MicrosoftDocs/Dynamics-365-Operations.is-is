# <a name="browser-requirements-for-dynamics-365-commerce-sites"></a>Vafrakröfur fyrir Dynamics 365 Commerce vefsvæði

[!include [banner](includes/banner.md)]

Þetta efnisatriði fjallar um vafrakröfur á Microsoft Dynamics 365 Commerce svæðum.

Dynamics 365 Commerce svæði eru studd best af eftirtöldum vefvöfrum. Farið á vefsvæði hugbúnaðarframleiðandans til að finna nýjustu útgáfu hvers vafra.

- [Microsoft Edge](https://www.microsoft.com/edge) (upp að núverandi útgáfu mínus 1)
- [Mozilla Firefox](https://www.mozilla.org/firefox/new) (upp að núverandi útgáfu mínus 1)
- [Google Chrome](https://www.google.com/chrome) (upp að núverandi útgáfu mínus 1)
- [Apple Safari](https://support.apple.com/downloads/safari) (upp að núverandi útgáfu mínus 1)

> [!NOTE]
> Internet Explorer er ekki stutt. Ef þú notar Internet Explorer er ekki víst að allir eiginleikar virki og upplifun notenda kann að vera skert.
