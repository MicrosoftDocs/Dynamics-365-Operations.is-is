# <a name="upload-files-other-than-images-and-videos"></a>Hlaða upp skrám öðrum en myndum og myndböndum

[!include [banner](includes/banner.md)]

Þessi grein lýsir því hvernig á að hlaða upp öðrum skrám en myndum og myndböndum inn Microsoft Dynamics 365 Commerce vefsmiður.

Margmiðlunarsafn vefsvæðishönnuðar Commerce styður upphleðslu tvöfaldra eigna annarra en mynda eða myndskeiða. Til dæmis gætirðu viljað hlaða inn Microsoft Excel, Microsoft Word, Microsoft PowerPoint eða PDF-skrám.

Eftirfarandi gerðir skjala eru studdar:
- 7Z
- AVI
- CS
- CSS
- DOC
- DOCX
- EPUB
- GIF
- INDD
- JAR
- JPG
- JPEG
- JS
- MP3
- MP4
- MPEG
- MPG
- ODP
- ODS
- ODT
- PDF
- PNG
- PPT
- PPTX
- PS
- QXP
- RAR
- RTF
- SVG
- TAR
- TGZ
- TXT
- WMV
- XLS
- XLSX
- XML
- ZIP

## <a name="upload-a-file"></a>Hlaða upp skrá

Til að hlaða upp skrá í vefsíðuhönnuð í Commerce skaltu fylgja þessum skrefum.

1. Í vinstri stýriglugganum velurðu **Margmiðlunarsafn**.
1. Veldu á skipanastikunni **Hlaða inn \> Hlaða upp margmiðlun**.
1. Veldu File Explorer eina eða fleiri skrár og veldu síðan **Opna**.
1. Í valmyndinni **Hlaða upp margmiðlun** slærðu inn titil, lýsingu og lýsigögn leitarorðs eftir þörfum.
1. Til að birta skár beint eftir uppflutning skaltu velja gátreitinn **Birta margmiðlunarefni eftir uppflutning**.
1. Veljið **Í lagi**.
