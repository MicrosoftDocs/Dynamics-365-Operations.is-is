## <a name="configure-pos-authentication-method"></a><span data-ttu-id="114f2-107">Sannvottunaraðferð sölustaðar skilgreind</span><span class="sxs-lookup"><span data-stu-id="114f2-107">Configure POS authentication method</span></span>

    - <span data-ttu-id="114f2-112">**Kenni og aðgangsorð starfsmanns** – Þessi sjálfgefni valkostur gerir kröfu um að notendur sölustaðar slái inn kenni og aðgangsorð starfsmanns til að skrá sig inn á sölustaðinn og fá aðgang að hnekkingaraðgerð stjórnanda.</span><span class="sxs-lookup"><span data-stu-id="114f2-112">**Personnel ID and Password** - This default option requires POS users to enter a personnel ID and password to sign in to the POS and to access manager override functionality.</span></span>
