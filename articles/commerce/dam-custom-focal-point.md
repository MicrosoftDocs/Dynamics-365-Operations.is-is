# <a name="customize-image-focal-points"></a><span data-ttu-id="cd93b-103">Sérstilla áherslupunkta myndar</span><span class="sxs-lookup"><span data-stu-id="cd93b-103">Customize image focal points</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="cd93b-104">Þetta efnisatriði lýsir því hvernig á að sérsníða áherslupunkta myndar í Microsoft Dynamics 365 Commerce svæðasmið.</span><span class="sxs-lookup"><span data-stu-id="cd93b-104">This topic describes how to customize image focal points in Microsoft Dynamics 365 Commerce site builder.</span></span>

<span data-ttu-id="cd93b-105">Þegar mynd er hlaðið upp í margmiðlunarsafn í vefsíðuhönnuði Commerce reynir kerfið að ákvarða þungamiðju myndarinnar.</span><span class="sxs-lookup"><span data-stu-id="cd93b-105">When an image is uploaded to the Commerce site builder Media Library, the system attempts to determine the focal point of the image.</span></span> <span data-ttu-id="cd93b-106">Til dæmis, ef myndin er af manneskju mun kerfið sjálfkrafa setja miðpunktinn á andlit viðkomandi.</span><span class="sxs-lookup"><span data-stu-id="cd93b-106">For example, if the image has a person on it, the system will set the focal point to the face of the person by default.</span></span> <span data-ttu-id="cd93b-107">Í flestum tilvikum virkar sjálfvirkur miðpunkturinn vel fyrir allar skoðunargáttir, en stundum gætirðu viljað stilla þungamiðjuna til að tryggja að ákveðinn hluti myndarinnar sé alltaf sýnilegur.</span><span class="sxs-lookup"><span data-stu-id="cd93b-107">In most cases the automatically set focal point works well for all viewports, but sometimes you may want to adjust the focal point to ensure that a specific part of the image is always visible.</span></span>

### <a name="define-a-custom-focal-point-for-an-image"></a><span data-ttu-id="cd93b-108">Skilgreindu sérsniðna þungamiðju fyrir mynd</span><span class="sxs-lookup"><span data-stu-id="cd93b-108">Define a custom focal point for an image</span></span>

<span data-ttu-id="cd93b-109">Fylgdu þessum skrefum til að skilgreina sérsniðna þungamiðju fyrir mynd.</span><span class="sxs-lookup"><span data-stu-id="cd93b-109">To define a custom focal point for an image, follow these steps.</span></span>

1. <span data-ttu-id="cd93b-110">Í vinstri flettiglugganum á vefsvæðishönnuði Commerce velurðu **Margmiðlunarsafn**.</span><span class="sxs-lookup"><span data-stu-id="cd93b-110">In the left navigation pane of Commerce site builder, select **Media Library**.</span></span>
1. <span data-ttu-id="cd93b-111">Í aðalglugganum velurðu myndina sem þú vilt breyta.</span><span class="sxs-lookup"><span data-stu-id="cd93b-111">In the main window, select the image you want to modify.</span></span>
1. <span data-ttu-id="cd93b-112">Á skipanastikunni velurðu **Breyta**.</span><span class="sxs-lookup"><span data-stu-id="cd93b-112">On the command bar, select **Edit**.</span></span>
1. <span data-ttu-id="cd93b-113">Veldu myndina sem á að fara inn í **Breyta stillingu**.</span><span class="sxs-lookup"><span data-stu-id="cd93b-113">Select the image to enter **Edit Mode**.</span></span>
1. <span data-ttu-id="cd93b-114">Undir **Breyta stillingu** velurðu **Breyta þungamiðju**.</span><span class="sxs-lookup"><span data-stu-id="cd93b-114">Under **Edit Mode**, select **Change Focal Point**.</span></span> <span data-ttu-id="cd93b-115">Hringlaga miðpunktur birtist yfir myndinni.</span><span class="sxs-lookup"><span data-stu-id="cd93b-115">A circular focal point control appears over the image.</span></span>
1. <span data-ttu-id="cd93b-116">Veldu miðpunktsstýringu til að færa hana yfir viðkomandi þungamiðju.</span><span class="sxs-lookup"><span data-stu-id="cd93b-116">Select the focal point control to move it over the desired focal point.</span></span>
1. <span data-ttu-id="cd93b-117">Þegar það er búið skaltu á skipanastikunni velja **Vista** og veldu síðan **Ljúka breytingu**.</span><span class="sxs-lookup"><span data-stu-id="cd93b-117">When you're done, on the command bar select **Save**, and then select **Finish editing**.</span></span>
