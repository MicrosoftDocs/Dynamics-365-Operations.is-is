# <a name="digital-asset-management-overview"></a>Yfirlit stafrænnar eignastýringar

[!include [banner](includes/banner.md)]

Þetta efni veitir yfirlit yfir stjórnun stafrænna eigna í vefsvæðishönnuði í Microsoft Dynamics 365 Commerce.

## <a name="overview"></a>Yfirlit

Margmiðlunarsafn Dynamics 365 Commerce vefsvæðishönnuðar er með ríkulega stafræna eignastýringarvirkni, þar á meðal stuðning við eftirfarandi:
- Mynd-eignir
- Vídeóeignir
- Aðrar tegundir tvöfaldra eigna, eins og Office-skjöl og PDF-skjöl
- Staðfærðar myndir
- Möppuupphleðslur
- Skurður á myndum
- Sérsnið á þungamiðju myndar
- Myndeignir Omni-rásar (vara, vöruafbrigði, verslun, flokkur, starfsmenn og viðskiptavini)

Stafrænar eignir eru rásarsértækar, að undanskildum eignum umrænna rásanna, sem eru sjálfgefnar umhverfisvænar. 

Stafrænar eignir og stjórnunaraðgerðir eru staðsettar innan byggingaraðila undir **Margmiðlunarsafn** í vinstri yfirlitsglugga á vefsvæði.
