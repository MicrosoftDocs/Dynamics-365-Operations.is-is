2. <span data-ttu-id="107c0-111">Veljið hvaða útlit sem er sem er með gerð forrits **Modern POS fyrir Windows** eða **Cloud POS**, og smellið síðan á **útlitshönnuður**.</span><span class="sxs-lookup"><span data-stu-id="107c0-111">Select any layout that has an application type of **Modern POS for Windows** or **Cloud POS**, and then click **Layout designer**.</span></span>

## <a name="troubleshoot-the-installation-of-the-layout-designer"></a><span data-ttu-id="107c0-119">Úrræðaleit vegna uppsetningar á útlitshönnuður</span><span class="sxs-lookup"><span data-stu-id="107c0-119">Troubleshoot the installation of the Layout designer</span></span>

- <span data-ttu-id="107c0-120">Þegar smellt er á **Hönnuður**, birtist ekki kvaðningin sækja (eða keyra) uppsetningarforrits eða gildandi öryggisstillingar leyfa þér ekki að sækja skrána.</span><span class="sxs-lookup"><span data-stu-id="107c0-120">When you click **Designer**, the prompt to download (or run) the installer doesn't appear, or your current security settings don't allow you to download the file.</span></span> 

    <span data-ttu-id="107c0-121">**Lausnir**</span><span class="sxs-lookup"><span data-stu-id="107c0-121">**Solutions:**</span></span>
