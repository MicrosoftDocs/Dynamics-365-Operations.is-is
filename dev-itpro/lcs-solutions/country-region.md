# <a name="localization-and-regulatory-features"></a>Staðfærslu og eftirlitseiginleikar

Microsoft Dynamics 356 aðgerða inniheldur virkni fyrir sumar lands-/ svæðisbundnum reglum og ferla. Þessi virkni er virkur byggt á aðalaðsetur lögaðilans virka. Þetta efnisatriði inniheldur lista yfir tilföng sem geta hjálpað til við að frekari upplýsingar um lands/svæðis-bundnar aðgerðir. 

## <a name="developer-and-isv-information"></a>Upplýsingar um forritara og ISV
Eftirfarandi gagnasöfn veita upplýsingar sem geta hjálpað til við forritara og ISVs sem eru að stofna lands/svæðis-bundnar sérsnið.
-   [Localization aðskilin virðislíkön](separate-localization-models.md)
-   [Nota tilliti lands/svæðis](apply-country-context.md)
-   [Reglubundnar verkalýðsfélaga](regulatory-certifications.md)
-   [Flokka localization aðgerðir](classify-localization-features.md)
-   [Kóðar lands - ISO 3166](https://www.iso.org/iso-3166-country-codes.html)

## <a name="localization-updates"></a>Uppfærir staðfærslu
Eftirfarandi gagnasöfn veita upplýsingar um staðfærslu áætlaðar og nýjar aðgerðir. 
-   [Staðfærslu portal](https://mbs.microsoft.com/customersource/northamerica/ax/support/support-news/GFMLocalizationPortalMC)
-   [Reglubundnar watch og samskipta reglubundnar uppfærslu](regulatory-watch-communication.md)
-   [Tilkynna Microsoft um land/svæði reglugerð viðvaranir og rakið stöðu reglubundnar aðgerðir](submit-localization-alerts.md)

## <a name="electronic-reporting"></a>Rafræn skýrslugerð
Rafræn skýrslugerð (ER) verkfæri gerir kleift að skilgreina snið fyrir rafræn skjöl samkvæmt lögboðnar kröfur í ýmsum löndum/svæðum. ER (rafræn skýrslugerð) gerir þér kleift að stjórna þessum sniðum í lífsferli þeirra. Frekari upplýsingar er að finna í eitt af eftirfarandi efnisatriði:
-   [Yfirlit yfir Rafræna skýrslugerð](../analytics/general-electronic-reporting.md)
-   [Stjórnun á lífsferli fyrir stillingar Rafrænnar skýrslugerðar](../analytics/general-electronic-reporting-manage-configuration-lifecycle.md)
-   [Rafræn skýrslugerð skilgreiningu](../analytics/electronic-reporting-configuration.md)
-   [Útvíkkuð yfir Electronic skýrslur aðgerða](../analytics/general-electronic-reporting-formulas-list-extension.md)
-   [Viðtökustaður rafrænnar skýrslugerðar](../analytics/electronic-reporting-destinations.md)
-   [Hlaða niður Grunnstillingar fyrir rafræna skýrslugerð úr Lifecycle Services](../analytics/download-electronic-reporting-configuration-lcs.md)
-   [Setja upp Rafræna skrá með Power BI](../analytics/general-electronic-reporting-report-configuration-get-data-powerbi.md)
