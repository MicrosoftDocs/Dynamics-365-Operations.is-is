# <a name="power-bi-content-recently-released-for-dynamics-365-for-operations"></a>Power BI efni nýlega gefið út fyrir Dynamics 365 fyrir Operations

[!include[banner](../includes/banner.md)]

"[!include[banner](../includes/banner.md)]"


Þetta efnisatriði útlistar efni Power BI sem hefur nýlega verið gefið út fyrir Dynamics 365 fyrir Operations.

<a name="power-bi-content-that-was-released-in-february-2017"></a>Power BI efnið sem var gefið út í í febrúar 2017
---------------------------------------------------

| Power BI-efni       | lýsing                                                                                                                                                                                                                                                                                                                                                                                   | Frekari upplýsingar                                                                                                         |
|------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| Afköst vöruhúss  | Þetta efni Microsoft Power BI hjálpar vöruhúsa- og verkstjórum að fylgjast með því sem kemur inn, fer út sem og mælikvörðum birgða. Notast er við vöruhúsakerfi, vörur og önnur færslugögn úr Microsoft Dynamics 365 fyrir Operators, og birtir bæði uppsöfnuð afköst vöruhúss og sundurliðun lánardrottna, vöruflokka og vörur, og svæði og vöruhús. | [Power BI-efni fjárhagslegrar frammistöðu](warehouse-power-bi-content.md) |
| Framleiðsluafköst | Þetta efni Power BI hjálpar framleiðslustjórum að fylgjast með gæðum framleiðsluaðgerða, með tilliti til tímabærrar framkvæmda, gæða og kostnaðar. Hún veitir bæði yfirlit yfir allt fyrirtækið eftir framleiðslu mælikvörðum og niðurbrot mælikvarða eftir vöru og tilföngum.                                                                                                            |                                                                                                                    |
