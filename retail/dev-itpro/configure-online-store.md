# <a name="configure-an-online-store"></a>Skilgreina netverslun

Þessi skrá veitir tengla í efnisatriði sem munu aðstoða miðlægt skilgreina og stjórna og netverslunar úr Microsoft Dynamics 365 fyrir Aðgerðir.

Efnisatriði eru í eftirfarandi töflu hjálp við að skilgreina Dynamics 365 aðgerða - íhluti Smásölu og smásölunetverslun í Dynamics 365 Aðgerðir biðlara.

## <a name="configure-an-online-store"></a>Skilgreina netverslun
| Verkefni                                                | Upplýsingar                                                                                                                                                                                                                                                                                                                                                   | Efni                                                                                                                                                                                                                                                                                                                                                                                                                                   |
|-----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Skilgreina íhluti Smásölu.                        | Setja upp og viðhalda upplýsingum um aðgerðir retail. Þessar upplýsingar innihalda verslanir, skatta, vörur, gjafakort, kynningartilboð og afslætti.                                                                                                                                                                                                          | [Uppsetning og umsjón með Retail](https://technet.microsoft.com/en-us/library/hh597201.aspx) (TechNet efni fyrir Microsoft Dynamics AX 2012)                                                                                                                                                                                                                                                                                          |
| Skilgreina yfirlitsstigveldi smásölurásar.    | Búa til tegund yfirlitsstigveldi smásölurásar til að setja upp uppbyggingu tegund fyrir afurðir sem bjóða gegnum netverslun. Hægt að skilgreina tegundastigveldi, og tengja afurðir eigindaflokkar afurðar og gildi afurðareiginda fyrir tegundum. Er að tengja tegundastigveldi netverslun.                            | [Setja upp smásölustigveldi](https://technet.microsoft.com/en-us/library/hh580593.aspx) (TechNet efni fyrir Microsoft Dynamics AX 2012) [Setja upp eigindir og eigindagerðir](https://technet.microsoft.com/en-us/library/hh227548.aspx) (TechNet efni fyrir Microsoft Dynamics AX 2012) [Setja upp eigindaflokka retail](https://technet.microsoft.com/en-us/library/jj728713.aspx) (TechNet efni fyrir Microsoft Dynamics AX 2012) |
| Bæta netverslun við stigveldi fyrirtækisins. | Áður en hægt er að úthluta úrvali afurða eða uppfylla fyrir netverslun sem þú stofnaðir eða mynda skýrslur sem innihalda upplýsingar frá sem netverslun, verður að úthluta verslun eina eða fleiri stigveldum. Að lágmarki þarf að úthluta netverslun á stigveldi fyrirtækis sem inniheldur afurðaúrval. | [Setja upp netverslun](https://technet.microsoft.com/en-us/library/jj682095.aspx) (TechNet efni fyrir Microsoft Dynamics AX 2012)                                                                                                                                                                                                                                                                                                     |
| Bæta afhendingarmáta við netverslun.          | Veljið afhendingaraðferðum sem býður upp á netverslun.                                                                                                                                                                                                                                                                                                 | [Setja upp netverslun](https://technet.microsoft.com/en-us/library/jj682095.aspx) (TechNet efni fyrir Microsoft Dynamics AX 2012)                                                                                                                                                                                                                                                                                                     |
| Varpa eigindum og bæta lýsigögnum við.                   | Veljið valkostir sem tilgreina hvernig eigindir fyrir hverja tegund eða rásar afurðar á behave í netversluninni á Microsoft SharePoint setrinu.                                                                                                                                                                                              | [Setja upp netverslun](https://technet.microsoft.com/en-us/library/jj682095.aspx) (TechNet efni fyrir Microsoft Dynamics AX 2012)                                                                                                                                                                                                                                                                                                     |
